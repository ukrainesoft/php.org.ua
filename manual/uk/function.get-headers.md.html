#get_headers

(PHP 5, PHP 7, PHP 8)

get_headers — Повертає всі заголовки з відповіді сервера на запит HTTP

Опис

get_headers(string $url, bool $associative = false, ?resource $context = null): array|false

get_headers() повертає масив із заголовками з відповіді сервера на HTTP-запит.

Список параметрів

url Цільовий URL.

associative Якщо необов'язковий параметр associative встановлено в ненульове значення, get_headers() розбере відповідь сервера та встановить ключі для повертається масиву.

context Коректний контекст ресурсу, створений за допомогою stream_context_create() або null, щоб використати контекст ресурсу за замовчуванням.

Значення, що повертаються

Повертає індексований або асоціативний масив із заголовками відповіді або false у разі виникнення помилки.

Список змін

Версія Опис
8.0.0 Тип параметра associative був змінений із цілого числа (int) на логічне значення (bool).
7.1.0 Доданий параметр context.
