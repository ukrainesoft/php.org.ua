yaml_emit

(PECL yaml >= 0.5.0)

yaml_emit — Повертає YAML-подання значення

Опис

yaml_emit( mixed $data, int $encoding = YAML_ANY_ENCODING, int $linebreak = YAML_ANY_BREAK, array $callbacks = null ): string

Повертає YAML-подання переданих у параметрі 'data' даних.

Список параметрів

data Дані data для кодування. Параметр може бути будь-якого типу, винятком ресурсу (resource).

encoding Вихідне кодування вибирається з YAML_ANY_ENCODING, YAML_UTF8_ENCODING, YAML_UTF16LE_ENCODING, YAML_UTF16BE_ENCODING.

linebreak Символ переведення рядка у виведенні YAML_ANY_BREAK, YAML_CR_BREAK, YAML_LN_BREAK, YAML_CRLN_BREAK.

callbacks Обробники контенту для створення вузлів YAML. Асоціативний масив (array), де як ключі використовуються імена класів, а як значень callback-функції (callable), які створюватимуть вузли для цих класів. Більше подробиць можна дізнатися у розділі про публікуючі callback-функції.

Значення, що повертаються

Повертає YAML-закодований рядок (string) у разі успішного виконання.

Список змін

Версія Опис
PECL yaml 1.1.0 Доданий параметр callbacks.
