IntlCalendar::clear

(PHP 5 = 5.5.0, PHP 7, PHP 8, PECL = 3.0.0a1)

IntlCalendar::clear — Очищає поле чи всі поля

Опис

Об'єктно-орієнтований стиль

public IntlCalendar::clear(?int $field = null): bool

Процедурний стиль

intlcal_clear(IntlCalendar $calendar, ?int $field = null): bool

Очищає всі поля, або певне поле. Очищене поле позначається як не задане, що дає йому найнижчий пріоритет щодо перевизначальним полям або навіть значенням за умовчанням під час обчислення часу. Крім того, його значення встановлено на 0, хоча, враховуючи низький пріоритет поля, його значення могло бути внутрішньо встановлено на інше значення на момент завершення запиту поля.

Список параметрів

calendar Примірник IntlCalendar.

field Одна з представлених у класі IntlCalendar констант полів типу дата час. Ціле число від 0 до IntlCalendar::FIELD_COUNT.

Значення, що повертаються

Функція завжди повертає true.
