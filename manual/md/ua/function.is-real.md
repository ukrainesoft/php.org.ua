#is_real

(PHP 4, PHP 5, PHP 7, PHP 8)

is_real - Псевдонім [is_float()](function.is-float.md)

### Опис

Ця функція є псевдонімом: [is_float()](function.is-float.md).

**Увага**

Псевдонім оголошений *УСТАРЕЛИМ* у PHP 7.4.0 та *Видалений* у PHP 8.0.0.
