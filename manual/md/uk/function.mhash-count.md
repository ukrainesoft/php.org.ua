#mhash_count

(PHP 4, PHP 5, PHP 7, PHP 8)

mhash_count — Отримати найбільш доступний ідентифікатор хешу

**Увага**

Функція оголошена *УСТАРНІЙ*, починаючи з PHP 8.1.0. Використати цю
функцію не рекомендується.

### Опис

**mhash_count**(): int

Повертає найбільш доступний ідентифікатор хешу.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Отримати найбільш доступний ідентифікатор хешу. Хеші нумеруються від 0
до цього ідентифікатора.

### Список змін

| Версія | Опис |
|--------|---------------------------------------- -------------------------------------------------- -|
| 8.1.0 | Функція оголошена застарілою. Використовуйте замість неї [функції `hash_*()`](ref.hash.md). |
