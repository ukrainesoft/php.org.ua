mysqli_stmt::$error_list

mysqli_stmt_error_list

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

mysqli_stmt::$error_list -- mysqli_stmt_error_list -- Повертає список помилок виконання останнього запиту

Опис

Об'єктно-орієнтований стиль

array $mysqli_stmt->error_list;

Процедурний стиль

mysqli_stmt_error_list(mysqli_stmt $statement): array

Повертає список помилок, що виникли під час виконання останньої операції над запитом, який може завершуватися успішно чи неуспішно.

Список параметрів

stmt Тільки для процедурного стилю: об'єкт mysqli_stmt, отриманий за допомогою mysqli_stmt_init().

Значення, що повертаються

Список помилок, кожна з яких представлена у вигляді масиву (array), містить код помилки, повідомлення про помилку, а також код стану sqlstate.
