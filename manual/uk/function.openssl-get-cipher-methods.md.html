openssl_get_cipher_methods

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

openssl_get_cipher_methods — Отримати список доступних алгоритмів шифрування

Опис

openssl_get_cipher_methods(bool $aliases = false): array

Повертає перелік доступних алгоритмів шифрування.

Список параметрів

aliases Якщо хочете включити псевдоніми шифрів у масив, що повертається, то встановіть цей параметр значення true.

Значення, що повертаються

Масив доступних методів шифрування. Зверніть увагу, що до OpenSSL 1.1.1 методи шифрування поверталися у верхньому та нижньому регістрі; з OpenSSL 1.1.1 повертаються лише малі варіанти.
