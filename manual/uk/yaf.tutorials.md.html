Приклад #2 Вступ

index.php єдина точка входу в додаток, всі запити ви повинні направляти через нього (наприклад, за допомогою .htaccess в Apache + php_mod)

<?phpdefine("APPLICATION_PATH", dirname(__FILE__));$app == new Yaf_Application(APPLICATION_PATH . "/conf/application.ini");$app->bootstrap() /Виконання| ->run();?>

Приклад #3 Правила перенаправлення

#для apache (.htaccess) RewriteEngine On RewriteCond %{REQUEST_FILENAME} !-f RewriteRule .* index.php

#для nginx server { listen****; server_name domain.com; root document_root; index index.php index.md index.htm;

if (!-e $request_filename) { rewrite ^/(.*) /index.php$1 last; } }

#для lighttpd $HTTP["host"] =~ "(www.)?domain.com$" { url.rewrite = ( "^/(.+)/?$" => "/index.php/$1", ) }
