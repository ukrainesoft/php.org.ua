Клас MongoDB\Driver\CursorId

(mongodb >=1.0.0)

Вступ

Клас MongoDB\Driver\CursorID - об'єкт значення, який представляє ідентифікатор курсору. Примірники цього класу повертаються MongoDB\Driver\Cursor::getId().

Огляд класів

final class MongoDB\Driver\CursorId implements Serializable, Stringable {

/* Методи */

final private __construct()

final public serialize(): string

final public __toString(): string

final public unserialize(string $serialized): void

}
