Мережеві функції

Зміст

  • checkdnsrr — Перевірка записів DNS, відповідні переданому імені вузла Інтернету або IP-адреси
  • closelog — Закриває з'єднання з системним журналом
  • dns_check_record - Псевдонім checkdnsrr
  • dns_get_mx - Псевдонім getmxrr
  • dns_get_record - Отримання ресурсних записів DNS хоста
  • fsockopen — Відкриває з'єднання з інтернет-сокетом або доменним сокетом Unix
  • gethostbyaddr — Отримує доменне ім'я хоста, що відповідає переданій IP-адресі
  • gethostbyname — Отримує IPv4-адресу, відповідний переданому імені хоста
  • gethostbynamel — Отримує список IPv4-адрес, що відповідають переданому доменному імені хоста
  • gethostname — Отримує ім'я хоста
  • getmxrr — Отримує записи MX, відповідні переданому доменному імені хоста
  • getprotobyname — Отримує номер протоколу на ім'я
  • getprotobynumber — Отримує ім'я протоколу за номером
  • getservbyname — Отримує номер порту, пов'язаного з інтернет-службою та протоколом
  • getservbyport — Отримує інтернет-службу, що відповідає заданому порту та протоколу
  • header_register_callback - Викликає функцію заголовка
  • header_remove — Видаляє раніше встановлені заголовки
  • header — Надсилання заголовка HTTP
  • headers_list — Повертає список переданих заголовків (або готових до відправлення)
  • headers_sent — Перевіряє, чи були вони надіслано заголовки
  • http_response_code - Отримує або встановлює код відповіді HTTP
  • inet_ntop — Конвертує упакований інтернет-адреса в формат, що читається
  • inet_pton - Конвертує читаний IP-адреса в його упаковані in_addr
  • ip2long — Конвертує рядок, який містить IPv4-адреса в ціле число
  • long2ip - Конвертує ціле число в IPv4-адреса
  • net_get_interfaces — Отримує мережеві інтерфейси
  • openlog — Відкриває підключення до системного журналу
  • pfsockopen — Відкриває постійне з'єднання з інтернет-сокетом або доменним сокетом Unix
  • setcookie — Надсилає cookie
  • setrawcookie — Надсилає cookie без URL-кодування значення
  • socket_get_status - Псевдонім stream_get_meta_data
  • socket_set_blocking - Псевдонім stream_set_blocking
  • socket_set_timeout - Псевдонім stream_set_timeout
  • syslog — Генерує повідомлення для системного журналу
