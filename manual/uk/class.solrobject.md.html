Клас SolrObject

(PECL solr > = 0.9.2)

Вступ

Об'єкт, властивості якого можна отримати за допомогою синтаксису масиву. Усі його властивості доступні лише читання.

Огляд класів

final class SolrObject implements ArrayAccess {

/* Методи */

public __construct()

public getPropertyNames(): array

public offsetExists(string $property_name): bool

public offsetGet(string $property_name): mixed

public offsetSet(string $property_name, string $property_value): void

public offsetUnset(string $property_name): void

public __destruct()

}

Зміст

  • SolrObject::__construct - Створює об'єкт Solr
  • SolrObject::__destruct - Деструктор
  • SolrObject::getPropertyNames - Повертає масив усіх імен властивостей
  • SolrObject::offsetExists - Перевіряє, чи існує властивість
  • SolrObject::offsetGet — Використовується для отримання властивості
  • SolrObject::offsetSet — Встановлює значення властивості
  • SolrObject::offsetUnset — Скидає значення властивості
