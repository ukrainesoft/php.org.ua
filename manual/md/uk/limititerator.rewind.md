# LimitIterator::rewind

(PHP 5 \>u003d 5.1.0, PHP 7, PHP 8)

LimitIterator::rewind — Переміщує курсор на початкову позицію

### Опис

public **LimitIterator::rewind**(): void

Переміщує ітератор на початкову позицію, задану під час створення об'єкта
конструктором
[LimitIterator::\_\_construct()](limititerator.construct.md).

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Функція не повертає значення після виконання.
