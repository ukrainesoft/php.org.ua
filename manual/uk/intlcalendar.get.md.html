IntlCalendar::get

(PHP 5 = 5.5.0, PHP 7, PHP 8, PECL = 3.0.0a1)

IntlCalendar::get — Отримує значення поля

Опис

Об'єктно-орієнтований стиль

public IntlCalendar::get(int $field): int|false

Процедурний стиль

intlcal_get(IntlCalendar $calendar, int $field): int|false

Отримує значення певного поля.

Список параметрів

calendar Примірник IntlCalendar.

field Одна з представлених у класі IntlCalendar констант полів типу дата час. Ціле число від 0 до IntlCalendar::FIELD_COUNT.

Значення, що повертаються

Ціле число із значенням поля часу.
