Робота з помилками XML

Робота з помилками XML під час завантаження документів дуже проста завданням. Використання функціональності libxml дозволяє придушити всі XML-помилки під час завантаження документа і потім обробити їх.

Об'єкт libXMLError, що повертається libxml_get_errors(), містить кілька властивостей, у тому числі повідомлення, номер рядки та колонку (позицію) цієї помилки.

Приклад #1 Завантаження синтаксично неправильного XML-рядка

<?phplibxml_use_internal_errors(true);$sxe = simplexml_load_string("<?xml version='1.0'><broken><xml></broken>");if (!$sxe) {    echo "Помилка " ";   foreach(libxml_get_errors() as $error) {        echo " ",,$error->message;    }}?>

Результат виконання цього прикладу:

Помилка завантаження XML Blank needed here parsing XML declaration: '?>' expected Opening and ending tag mismatch: xml line 1 and broken Premature end of data in tag broken line 1
