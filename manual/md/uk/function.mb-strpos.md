#mb_strpos

(PHP 4 \>u003d 4.0.6, PHP 5, PHP 7, PHP 8)

mb_strpos — Пошук позиції першого входження одного рядка до іншого

### Опис

**mb_strpos**(
string `$haystack`,
string `$needle`,
int `$offset` u003d 0,
?string `$encoding` u003d **`null`**
): int\|false

Шукає позицію першого входження одного рядка string до іншого рядка
string.

Виконує безпечну з погляду багатобайтних кодувань операцію
[strpos()](function.strpos.md), яка спирається на число символів у
рядку. Перший символ стоїть на позиції 0, позиція другого 1 і таке інше.

### Список параметрів

`haystack`
Рядок string, в якій проводиться пошук.

`needle`
Рядок, пошук якого проводиться в рядку `haystack`. На відміну від
[strpos()](function.strpos.md), числові значення не застосовуються в
як порядкові номери символів.

`offset`
Зміщення початку пошуку. Якщо не встановлено, використовується 0. Якщо значення
негативно, відлік іде з кінця рядка.

`encoding`
Параметр 'encoding' є символьним кодуванням. Якщо він
опущений або дорівнює **`null`**, замість нього буде використано значення
внутрішнього кодування.

### Значення, що повертаються

Повертає число - позицію першого входження рядка `needle` у рядок
`haystack` string. Якщо рядок `needle` не знайдено, функція поверне
**`false`**.

### Список змін

| Версія | Опис |
|--------|---------------------------------------- -------------------------|
| 8.0.0 | `needle` тепер приймає порожній рядок. |
| 8.0.0 | Тепер параметр `encoding` може набувати значення **`null`**. |
| 7.1.0 | Додано підтримку негативних значень `offset`. |
