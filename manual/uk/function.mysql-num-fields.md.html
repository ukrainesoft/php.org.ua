mysql_num_fields

(PHP 4, PHP 5)

mysql_num_fields — Повертає кількість полів результату запиту

Увага

Цей модуль застарів, починаючи з версії PHP 5.5.0, і вилучений до PHP 7.0.0. Використовуйте замість нього MySQLi або PDO_MySQL. Дивіться також інструкцію MySQL: вибір API. Альтернативи для цієї функції:

  • mysqli_num_fields()
  • PDOStatement::columnCount()

Опис

mysql_num_fields(resource $result): int|false

Повертає кількість полів у результаті запиту.

Список параметрів

result Оброблюваний результат запита. Цей результат може бути отриманий за допомогою функції mysql_query().

Значення, що повертаються

Повертає кількість полів у результаті запиту (resource) у разі успішного виконання або false у разі виникнення помилки.

Приклади

Приклад #1 Приклад використання mysql_num_fields()

<?php$result = mysql_query("SELECT id,email FROM people WHERE id = '42'");if (!$result) {    echo 'Не удалося виконати запрос: . mysql_error(); exit;}/* повертає 2, так як id,email === двом полям */echo mysql_num_fields($result);?>

Примітки

Примітка:

Для зворотної сумісності може бути використаний наступний застарілий псевдонім: mysql_numfields()

Дивіться також

  • mysql_select_db() - Вибирає базу даних MySQL
  • mysql_query() - Надсилає запит MySQL
  • mysql_fetch_field() - Повертає інформацію про колонку з результату запиту як об'єкта
  • mysql_num_rows() - Повертає кількість рядів результату запиту
