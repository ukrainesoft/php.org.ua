mysql_fetch_lengths

(PHP 4, PHP 5)

mysql_fetch_lengths — Повертає довжину кожного поля в результаті

Увага

Цей модуль застарів, починаючи з версії PHP 5.5.0, і вилучений до PHP 7.0.0. Використовуйте замість нього MySQLi або PDO_MySQL. Дивіться також інструкцію MySQL: вибір API. Альтернативи для цієї функції:

  • mysqli_fetch_lengths()
  • PDOStatement::getColumnMeta()

Опис

mysql_fetch_lengths(resource $result): array|false

Повертає масив довжин для кожного поля, що міститься в останньому ряду результату, отриманому з MySQL.

mysql_fetch_lengths() повертає довжини кожного поля, що міститься в останньому ряду, обробленому функціями mysql_fetch_row(), mysql_fetch_assoc(), mysql_fetch_array() та mysql_fetch_object() у масиві, починається з 0.

Список параметрів

result Оброблюваний результат запита. Цей результат може бути отриманий за допомогою функції mysql_query().

Значення, що повертаються

Масив (array) довжин у разі успішного виконання або false у разі виникнення помилки.

Приклади

Приклад #1 Приклад використання mysql_fetch_lengths()

<?php$result = mysql_query("SELECT id,email FROM people WHERE id = '42'");if (!$result) {    echo 'Помилка виконання запиту: mysql_error(); exit;}$row     = mysql_fetch_assoc($result);$lengths = mysql_fetch_lengths($result);print_r($row);print_r($lengths);?>

Результатом виконання цього прикладу буде щось подібне:

Array ( [id] => 42 [email] => user@example.com ) Array ( [0] => 2 [1] => 16 )

Дивіться також

  • mysql_field_len() - Повертає довжину вказаного поля
  • mysql_fetch_row() - Обробляє ряд результату запиту та повертає масив з числовими індексами
  • strlen() - Повертає довжину рядка
