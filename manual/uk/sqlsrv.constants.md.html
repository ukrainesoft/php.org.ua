Зумовлені константи

Наведені нижче константи визначені даним модулем і можуть бути доступні лише в тому випадку, якщо PHP був зібраний за допомогою цього модуля або в тому випадку, якщо даний модуль був динамічно завантажений під час виконання.

SQLSRV_FETCH_ASSOC (int) Переданий як параметр, змушує функцію sqlsrv_fetch_array() повертати асоціативний масив.

SQLSRV_FETCH_NUMERIC (int) Переданий як параметр, змушує функцію sqlsrv_fetch_array() повертати масив із числовими ключами.

SQLSRV_FETCH_BOTH (int) Переданий як параметр, змушує функцію sqlsrv_fetch_array() повертати масив з ключами та числовими та асоціативними (поведінка за замовчуванням).

SQLSRV_ERR_ALL (int) Переданий як параметр, змушує функцію sqlsrv_errors() повертати і помилки та попередження (за замовчуванням).

SQLSRV_ERR_ERRORS (int) Переданий як параметр, змушує функцію sqlsrv_errors() повертати лише помилки (без попереджень).

SQLSRV_ERR_WARNINGS (int) Переданий як параметр, змушує функцію sqlsrv_errors() повертати тільки попередження (без помилок).

SQLSRV_LOG_SYSTEM_ALL (int) Переданий як параметр функції sqlsrv_configure(), включає журналування всіх підсистем.

SQLSRV_LOG_SYSTEM_CONN (int) Переданий як параметр функції sqlsrv_configure(), включає журналування активності з'єднання.

SQLSRV_LOG_SYSTEM_INIT (int) Переданий як параметр функції sqlsrv_configure(), включає журналювання активності ініціалізації

SQLSRV_LOG_SYSTEM_OFF (int) Переданий як параметр функції sqlsrv_configure(), відключає журналування всіх підсистем.

SQLSRV_LOG_SYSTEM_STMT (int) Переданий як параметр функції sqlsrv_configure(), включає журналювання активності виконання виразів (statement activity).

SQLSRV_LOG_SYSTEM_UTIL (int) Переданий як параметр функції sqlsrv_configure(), включає журналування активності функцій помилок

SQLSRV_LOG_SEVERITY_ALL (int) Переданий як параметр функції sqlsrv_configure(), визначає, що помилки, попередження та зауваження журналюватимуться.

SQLSRV_LOG_SEVERITY_ERROR (int) Переданий як параметр функції sqlsrv_configure(), визначає, що помилки журналюватимуться.

SQLSRV_LOG_SEVERITY_NOTICE (int) Переданий як параметр функції sqlsrv_configure(), визначає, що зауваження журналюватимуться.

SQLSRV_LOG_SEVERITY_WARNING (int) Переданий як параметр функції sqlsrv_configure(), визначає, що попередження журналюватимуться.

SQLSRV_NULLABLE_YES (int) Показує, що значення шпальти може бути NULL.

SQLSRV_NULLABLE_NO (int) Показує, що значення стовпця може бути NULL.

SQLSRV_NULLABLE_UNKNOWN (int) Показує, що невідомо, чи значення стовпця може бути NULL.

SQLSRV_PARAM_IN (int) При передачі як параметр функції sqlsrv_query() або sqlsrv_prepare() показує, що параметр є вхідним.

SQLSRV_PARAM_INOUT (int) При передачі як параметр функції sqlsrv_query() або sqlsrv_prepare() показує, що параметр є вхідним і вихідним.

SQLSRV_PARAM_OUT (int) При передачі як параметр функції sqlsrv_query() або sqlsrv_prepare() показує, що параметр є вихідним.
