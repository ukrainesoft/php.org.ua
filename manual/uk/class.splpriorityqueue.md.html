Клас SplPriorityQueue

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

Вступ

Клас SplPriorityQueue забезпечує основні функціональні можливості пріоритетної черги, реалізований за допомогою купи з максимальним елементом зверху (max-heap).

Примітка: Порядок елементів з однаковим пріоритетом не визначено. Він може відрізнятись від порядку, в якому елементи були Вставлені.

Огляд класів

class SplPriorityQueue implements Iterator, Countable {

/* Методи */

public compare(mixed $priority1, mixed $priority2): int

public count(): int

public current(): mixed

public extract(): mixed

public getExtractFlags(): int

public insert(mixed $value, mixed $priority): bool

public isCorrupted(): bool

public isEmpty(): bool

public key(): int

public next(): void

public recoverFromCorruption(): bool

public rewind(): void

public setExtractFlags(int $flags): int

public top(): mixed

public valid(): bool

}

Зміст

  • SplPriorityQueue::compare - Порівнює пріоритети для коректного розміщення елементів у чергу
  • SplPriorityQueue::count — Виробляє підрахунок елементів у черзі
  • SplPriorityQueue::current - Повертає поточний вузол, який вказує ітератор
  • SplPriorityQueue::extract - Витягує вузол з початку черги та пересортує її
  • SplPriorityQueue::getExtractFlags — Отримати прапори вилучення
  • SplPriorityQueue::insert — Додає елемент у чергу і пересортує її
  • SplPriorityQueue::isCorrupted - Вказує, чи є пріоритетна черга у пошкодженому стані
  • SplPriorityQueue::isEmpty - Перевіряє, чи є черга порожньою
  • SplPriorityQueue::key — Повертає індекс поточного вузла
  • SplPriorityQueue::next — Перехід до наступному вузлу
  • SplPriorityQueue::recoverFromCorruption
  • Відновлює коректний стан черги
  • SplPriorityQueue::rewind — Перекладає ітератор на початок черги
  • SplPriorityQueue::setExtractFlags — Встановлює режим вилучення вузлів
  • SplPriorityQueue::top — Повертає вузол, що знаходиться на початку черги
  • SplPriorityQueue::valid — Перевіряє, чи є в черзі ще елементи
