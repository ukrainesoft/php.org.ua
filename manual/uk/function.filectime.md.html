filectime

(PHP 4, PHP 5, PHP 7, PHP 8)

filectime — Повертає час зміни індексного дескриптора файлу

Опис

filectime(string $filename): int|false

Повертає час зміни індексного дескриптора (inode) файлу.

Список параметрів

filename Шлях до файлу.

Значення, що повертаються

Повертає час останньої зміни файлу або false у разі виникнення помилки. Час повертається у форматі тимчасової мітки Unix.

Помилки

У разі невдалого завершення роботи генерується помилка рівня E_WARNING.

Приклади

Приклад #1 Приклад використання функції filectime()

<?php// Приклад висновку: Файл somefile.txt в останній раз був змінений: December 29 2002 22:16:23.$filename = 'somefile.txt';  filename в останній раз був змінений: "". date("F d Y H:i:s.", filectime($filename));}?>

Примітки

Примітка:

Примітка. На більшості платформ Unix файл вважається зміненим, якщо змінено дані його індексного дескриптора, що включає інформацію про права на файл, про його власника, групу та інші метадані, що містяться в індексному дескрипторі. Зверніться також до опис функцій filemtime() (дана Функція корисна для створення повідомлень типу: "Останнє оновлення від ..." на веб-сторінках) та fileatime().

Примітка:

Врахуйте також, що деякі описи роботи Unix ctime представляється як час створення файлу. Це не вірно. У більшості файлових систем Unix поняття часу створення файлу відсутнє.

Примітка:

Врахуйте, що обробка часу може відрізнятися в різних файлових системах.

Примітка: Результати цієї функції кешуються. Детальніше інформацію дивіться у розділі clearstatcache().
