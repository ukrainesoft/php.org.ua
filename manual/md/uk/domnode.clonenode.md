# DOMNode::cloneNode

(PHP 5, PHP 7, PHP 8)

DOMNode::cloneNode — Клонує вузол

### Опис

public **DOMNode::cloneNode**(bool `$deep` u003d **`false`**):
[DOMNode](class.domnode.md)\|false

Створює копію вузла.

### Список параметрів

`deep`
Вказує, чи потрібно копіювати нащадків. Цей параметр за замовчуванням
має значення **`false`**.

### Значення, що повертаються

Копія вузла.
