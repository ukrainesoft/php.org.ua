SoapClient::__setLocation

(PHP 5 >= 5.0.4, PHP 7, PHP 8)

SoapClient::__setLocation — Встановлює адресу використовуваної веб-служби

Опис

public SoapClient::__setLocation(?string $location = null): ?string

Встановлює URL кінцевої точки, що зачіпає наступні SOAP-запити. Еквівалентна настройці location, що вказується в конструктори SoapClient.

Примітка:

Виклик методу необов'язковий. За замовчуванням SoapClient використовує адресу з WSDL-файл.

Список параметрів

location Новий URL кінцевої точки.

Значення, що повертаються

Старий URL кінцевої точки.

Список змін

Версія Опис
8.0.3 location тепер допускає значення null.

Приклади

Приклад #1 Приклад використання SoapClient::__setLocation()

<?php$client = new SoapClient('http://example.com/webservice.php?wsdl');$client->__setLocation('http://www.somethirdparty.com');$old_location = $ client->__setLocation(); // скидає налаштування адресаecho $old_location;?>

Результатом виконання цього прикладу буде щось подібне:

http://www.somethirdparty.com
