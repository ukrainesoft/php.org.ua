Приклади

У документації по Reflection ви зможете зустріти безліч прикладів, найчастіше в описі конструкторів класів.

Приклад #1 Приклад використання Reflection із shell (терміналу)

`` shellcode $ php --rf strlen $ php --rc finfo $ php --re json $ php --ri dom
