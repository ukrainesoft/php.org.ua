gzinflate

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gzinflate — Розпакувати стислий рядок

Опис

gzinflate(string $data, int $max_length = 0): string|false

Розпаковує стислий рядок.

Список параметрів

data Дані, стиснуті функцією gzdeflate().

max_length Максимальний розмір даних, що декодуються.

Значення, що повертаються

Розпаковані дані або false у разі виникнення помилки.

Функція поверне помилку, якщо стиснуті дані в 32768 разів більші за розмір. стислих даних data або, якщо max_length не 0 і більше, ніж необов'язковий параметр max_length.
