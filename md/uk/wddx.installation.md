## Установка

## PHP 7.4

Цей модуль був переміщений до репозиторію [PECL](https://pecl.php.net/)
і більше не постачається з PHP 7.4.0

Інформація щодо встановлення цього модуля PECL може бути знайдена у розділі
керівництва [Встановлення модулів PECL](install.pecl.md). Додаткова
інформація, така як нові версії, завантаження, вихідні файли,
інформація про розробника та CHANGELOG, може бути знайдена тут:
[»https://pecl.php.net/package/wddx](https://pecl.php.net/package/wddx).

## PHP \< 7.4

Після встановлення необхідної бібліотеки expat, скомпілюйте PHP з
**--enable-wddx** та використовуйте **--with-libexpat-dir** для expat.

Версія PHP для Windows має вбудовану підтримку цього модуля. Це
означає, що для використання цих функцій не потрібне завантаження
жодних додаткових модулів.
