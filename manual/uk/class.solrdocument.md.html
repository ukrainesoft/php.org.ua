Клас SolrDocument

(PECL solr > = 0.9.2)

Вступ

Подає документ Solr, отриманий з відповіді запит.

Зумовлені константи

SolrDocument::SORT_DEFAULT Стандартний режим для сортування полів у документі.

SolrDocument::SORT_ASC Сортує поля у порядку зростання

SolrDocument::SORT_DESC Сортує поля в порядку зменшення

SolrDocument::SORT_FIELD_NAME Сортує поля на ім'я поля.

SolrDocument::SORT_FIELD_VALUE_COUNT Сортує поля за кількістю значень у кожному полі.

SolrDocument::SORT_FIELD_BOOST_VALUE Сортує поля за значенням посилення.
