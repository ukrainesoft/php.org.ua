Функції xdiff

Зміст

  • xdiff_file_bdiff_size - Отримати розмір файлу після застосування бінарного патчу
  • xdiff_file_bdiff — Створити бінарний патч порівнюючи два файли
  • xdiff_file_bpatch — Застосувати бінарний патч до файлу
  • xdiff_file_diff_binary - Псевдонім для xdiff_file_bdiff
  • xdiff_file_diff — Створити уніфікований патч із порівняння двох файлів
  • xdiff_file_merge3 — Об'єднання трьох файлів в один
  • xdiff_file_patch_binary - Псевдонім для xdiff_file_bpatch
  • xdiff_file_patch — Застосування уніфікованого патчу до файлу
  • xdiff_file_rabdiff — Створити бінарний патч порівнюючи два файли за допомогою поліномінального алгоритму Rabin fingerprinting
  • xdiff_string_bdiff_size - Отримати розмір файлу, створеного після застосування бінарного патча
  • xdiff_string_bdiff — Створити бінарний патч для двох рядків
  • xdiff_string_bpatch - Застосування бінарного патча до рядка
  • xdiff_string_diff_binary - Псевдонім для xdiff_string_bdiff
  • xdiff_string_diff — Створити звичайний патч для двох рядків
  • xdiff_string_merge3 - Об'єднати три рядки в один
  • xdiff_string_patch_binary
  • Псевдонім для xdiff_string_bpatch
  • xdiff_string_patch — Застосувати звичайний патч до рядка
  • xdiff_string_rabdiff - Порівняти два рядки і створити бінарний патч використовуючи поліномінальний алгоритм Rabin fingerprint
