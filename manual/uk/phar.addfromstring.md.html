Phar::addFromString

(PHP 5 = 5.3.0, PHP 7, PHP 8, PECL phar = 2.0.0)

Phar::addFromString — Додає в phar-архів файл з рядка

Опис

public Phar::addFromString(string $localName, string $contents): void

Примітка:

Для коректної роботи з об'єктами Phar цим методом необхідна установка значення php.ini phar.readonly у 0. В інакше, буде викинуто виняток PharException.

За допомогою цього методу в phar-архів може бути доданий будь-який рядок. Файл буде збережено в архіві під ім'ям, вказаним у параметрі localname. Цей метод аналогічний ZipArchive::addFromString().

Список параметрів

localName Шлях, яким файл буде збережено в архіві.

contents Вміст файлу для збереження.

Значення, що повертаються

Немає значення, що повертається, у разі виникнення помилки викидається виняток.

Приклади

Приклад #1 Приклад використання Phar::addFromString()

Примітки

Примітка: Phar::addFile(), Phar::addFromString() і Phar::offsetSet() зберігає новий phar-архів щоразу під час їх виклику. Якщо продуктивність викликає занепокоєння, натомість слід використовувати Phar::buildFromDirectory() або Phar::buildFromIterator().

Дивіться також

  • Phar::offsetSet() - Зміна вмісту файлу
  • PharData::addFromString() - Створити файл із заданим вмістом у tar/zip-архіві
  • Phar::addFile() - Додає в phar-архів файл з файлової системи
  • Phar::addEmptyDir() - Додає до phar-архів порожню директорію
