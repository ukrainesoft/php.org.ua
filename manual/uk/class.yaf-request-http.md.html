Клас Yaf_Request_Http

(Yaf >=1.0.0)

Вступ

Будь-який запит від клієнта ініціалізується як Yaf_Request_Http. Ви можете отримати інформацію про запит, наприклад, URI та параметри запиту, використовуючи методи цього.

Примітка:

З метою безпеки $_GET/$_POST доступні тільки для читання в Yaf, що означає, що якщо ви встановите значення для цих глобальних змінних, ви не зможете отримати їх за допомогою Yaf_Request_Http::getQuery() або Yaf_Request_Http::getPost().

Якщо виникає необхідність використання такого функціоналу при модульному тестуванні, Yaf може бути зібраний за допомогою --enable-yaf-debug, який дозволить Yaf прочитати значення, задане Користувачем через скрипт.

У цьому випадку Yaf видасть попередження E_STRICT, щоб нагадати вам про це: Strict Standards: you are running yaf in debug mode
