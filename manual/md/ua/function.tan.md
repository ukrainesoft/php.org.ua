#tan

(PHP 4, PHP 5, PHP 7, PHP 8)

tan - Тангенс

### Опис

**tan**(float `$num`): float

**tan()** повертає тангенс параметра `num`. Параметр `num` задається в
радіанах.

### Список параметрів

`num`
Значення у радіанах

### Значення, що повертаються

Тангенс кута `num`

### Приклади

**Приклад #1 Приклад використання **tan()****

`<?phpecho tan(M_PI_4); // 1?> `
