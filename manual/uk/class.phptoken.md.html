Клас PhpToken

(PHP 8)

Вступ

Цей клас надає альтернативу функції token_get_all(). Тоді як функція повертає токени або у вигляді односимвольного рядка, або у вигляді масиву з ідентифікатором токена, його текстом та номером рядка, PhpToken::tokenize() нормалізує всі токени в об'єкти PhpToken, що дозволяє набагато зручніше працювати з токенами.

Огляд класів

class PhpToken implements Stringable {

/* Властивості */

public int $id;

public string $text;

public int $line;

public int $pos;

/* Методи */

final public __construct( int $id, string $text, int $line = -1, int $pos = -1 )

public getTokenName(): ?string

public is(int|string|array $kind): bool

public isIgnorable(): bool

public __toString(): string

public static tokenize(string $code, int $flags = 0): array

}

Властивості

id Одна з констант T_*, або символ ASCII, що є односимвольним токен.

text Текстове вміст токена.

line Номер рядка (починаючи з 1), з якого починається токен.

pos Початкова позиція (починаючи з 0) токена у рядку.

Зміст

  • PhpToken::__construct — Створює об'єкт PhpToken
  • PhpToken::getTokenName — Повертає ім'я токена
  • PhpToken::is — Перевіряє, чи відповідає токен зазначеного типу
  • PhpToken::isIgnorable — Повідомляє, буде чи токен ігноруватися парсером PHP
  • PhpToken::__toString - Повертає текстовий вміст токена
  • PhpToken::tokenize - Розбирає задану рядок, що містить програму на PHP, на масив об'єктів PhpToken
