mysql_data_seek

(PHP 4, PHP 5)

mysql_data_seek — Переміщує внутрішній покажчик у результаті запиту

Увага

Цей модуль застарів, починаючи з версії PHP 5.5.0, і вилучений до PHP 7.0.0. Використовуйте замість нього MySQLi або PDO_MySQL. Дивіться також інструкцію MySQL: вибір API. Альтернативи для цієї функції:

  • mysqli_data_seek()
  • PDO::FETCH_ORI_ABS

Опис

mysql_data_seek(resource $result, int $row_number): bool

mysql_data_seek() переміщує внутрішній покажчик результату запиту, з яким пов'язаний переданий дескриптор, до ряду із зазначеним номером. Наступний дзвінок до функції отримання даних MySQL, такий як mysql_fetch_assoc(), поверне саме його.

Нумерація row_number починається з 0. row_number має бути значенням у діапазоні від 0 до mysql_num_rows() - 1. Однак, якщо результат порожній (mysql_num_rows() == 0), то спроба зсуву покажчика до нульового ряду завершиться невдачею – буде викликана помилка рівня E_WARNING та mysql_data_seek() поверне false.

Список параметрів

result Оброблюваний результат запита. Цей результат може бути отриманий за допомогою функції mysql_query().

row_number Бажаний номер ряду в отриманому дескрипторі результату.

Значення, що повертаються

Повертає true у разі успішного виконання або false у у разі виникнення помилки.

Примітки

Примітка:

Функція mysql_data_seek() може бути використана тільки з mysql_query(), але не з mysql_unbuffered_query().

Дивіться також

  • mysql_query() - Надсилає запит MySQL
  • mysql_num_rows() - Повертає кількість рядів результату запиту
  • mysql_fetch_row() - Обробляє ряд результату запиту та повертає масив з числовими індексами
  • mysql_fetch_assoc() - Повертає ряд результату запиту як асоціативний масив
  • mysql_fetch_array() - Обробляє ряд результатів запиту, повертаючи асоціативний масив, чисельний масив чи обидва
  • mysql_fetch_object() - Обробляє ряд результату запиту та повертає об'єкт
