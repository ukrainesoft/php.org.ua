Інтерфейс Traversable

(PHP 5, PHP 7, PHP 8)

Вступ

Інтерфейс, що визначає, чи є клас обхідним (traversable) з використанням foreach.

Абстрактний базовий інтерфейс, який не може бути реалізований сам собі. Натомість повинен реалізовуватися IteratorAggregate або Iterator.

Примітка:

Внутрішні (вбудовані) класи, які реалізують цей інтерфейс, можуть бути використані в конструкції foreach і зобов'язані реалізовувати IteratorAggregate або Iterator.

Примітка:

Це внутрішній інтерфейс, який не може бути реалізований у скрипті PHP. Замість нього потрібно використовувати або IteratorAggregate, або Iterator. При реалізації інтерфейсу успадкованого від Traversable, переконайтеся, що в секції implements перед його ім'ям стоїть IteratorAggregate або Iterator.

Огляд інтерфейсів

interface Traversable {

}

Цей інтерфейс не має методів, його єдина мета – бути базовим інтерфейсом для всіх обхідних класів.
