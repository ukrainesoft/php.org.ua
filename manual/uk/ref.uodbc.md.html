Функції ODBC

Зміст

  • odbc_autocommit — Перемикає поведінка автоматичної фіксації
  • odbc_binmode - Керує обробкою двійкових даних стовпця
  • odbc_close_all - Закриває все з'єднання ODBC
  • odbc_close — Закриває з'єднання ODBC
  • odbc_columnprivileges - Перелічує стовпці та пов'язані привілеї для даної таблиці
  • odbc_columns - Перераховує імена стовпців у зазначених таблицях
  • odbc_commit - Фіксує транзакцію ODBC
  • odbc_connect — З'єднує з джерелом даних
  • odbc_cursor — Повертає курсор.
  • odbc_data_source - Повертає інформацію про доступні DSN
  • odbc_do - Псевдонім odbc_exec
  • odbc_error — Повертає останній код помилки
  • odbc_errormsg - Повертає останнє повідомлення про помилку
  • odbc_exec — Виконує інструкцію SQL безпосередньо
  • odbc_execute — Виконує запит
  • odbc_fetch_array - Повертає рядок результату у вигляді асоціативного масиву
  • odbc_fetch_into - Повертає одну рядок результату в масив
  • odbc_fetch_object - Повертає рядок результату як об'єкта
  • odbc_fetch_row — Повертає рядок
  • odbc_field_len - Повертає довжину (точність) поля
  • odbc_field_name — Повертає ім'я стовпця
  • odbc_field_num — Повертає номер стовпця
  • odbc_field_precision - Псевдонім odbc_field_len
  • odbc_field_scale - Повертає масштаб поля
  • odbc_field_type — Повертає тип даних поля
  • odbc_foreignkeys - Повертає список зовнішніх ключів
  • odbc_free_result - Звільняє ресурси, пов'язані з результатом
  • odbc_gettypeinfo — Повертає інформацію про типи даних, що підтримуються джерелом даних
  • odbc_longreadlen - Обробляє стовпці LONG
  • odbc_next_result — Перевіряє, чи є кілька результатів
  • odbc_num_fields - Повертає кількість стовпців в результаті
  • odbc_num_rows — Повертає кількість рядків у результаті
  • odbc_pconnect — Відкриває постійне з'єднання з базою даних
  • odbc_prepare — Підготовка запиту до виконанню
  • odbc_primarykeys — Отримує первинні ключі таблиці
  • odbc_procedurecolumns - Отримує інформацію про параметри процедур
  • odbc_procedures — Отримує список процедур, що зберігаються у певному джерелі даних
  • odbc_result_all — Виводить результат у вигляді HTML-таблиці
  • odbc_result — Отримує дані результату
  • odbc_rollback - Відкочує транзакцію
  • odbc_setoption — Регулює налаштування ODBC
  • odbc_specialcolumns — Витягує особливі стовпці
  • odbc_statistics - Отримує статистику про таблицю
  • odbc_tableprivileges - Перелічує таблиці та привілеї, пов'язані з кожною таблицею
  • odbc_tables — Отримує список імен таблиць, що зберігаються у певному джерелі даних
