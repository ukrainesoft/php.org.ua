#call_user_func

(PHP 4, PHP 5, PHP 7, PHP 8)

call_user_func — Викликає callback-функцію, задану у першому параметрі

Опис

call_user_func(callable $callback, mixed ...$args): mixed

Викликає callback-функцію, передану першим параметром, та передає Інші параметри як аргументи. callback.

Список параметрів

callback Функція типу callable.

args Нуль або більше параметрів, що передаються в callback-функцію.

Примітка:

Врахуйте, що параметри для call_user_func() не передаються за Посилання.

Приклад #1 Приклад використання call_user_func() та посилання

Значення, що повертаються

Повертає значення, повернене функцією callback.

Примітки

Примітка:

Callback-функції, зареєстровані такими функціями як call_user_func() та call_user_func_array(), не будуть викликані за наявності не спійманого виключення, кинутого в попередній callback-функції.
