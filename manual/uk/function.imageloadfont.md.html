(PHP 4, PHP 5, PHP 7, PHP 8)

imageloadfont — Завантаження шрифту

Опис

imageloadfont(string $filename): GdFont|false

imageloadfont() завантажує певний користувачем шрифт та повертає його ідентифікатор.

Список параметрів

filename Формат файлу шрифту є двійковим і залежить від архітектури системи. Це означає необхідність генерувати файл шрифту на тому ж процесорі, який працює PHP.

позиція байта тип даних C опис
байти 0-3 int кількість символів у шрифті
байти 4-7 int значення першого символу у шрифті (часто 32 - пробіл)
байти 8-11 int ширина пікселя кожного символу
байти 12-15 int висота пікселя кожного символу
байти 16 - char масив із даними символів, один байт на піксел у кожному символі. Для всіх кількість висота широта байт.

Формат файлу шрифту

Значення, що повертаються

Повертає екземпляр GdFont або false у разі виникнення помилки. Ідентифікатор шрифту, який завжди більший 5 для запобігання конфліктам із вбудованими шрифтами, або false у разі виникнення помилки.

Список змін

Версія Опис
8.1.0 Повертає екземпляр GdFont; раніше поверталося ціле число (int).
