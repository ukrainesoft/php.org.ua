decoct

(PHP 4, PHP 5, PHP 7, PHP 8)

decoct - Переводить число з десяткової системи числення у вісімкову

Опис

decoct(int $num): string

Повертає рядок, що містить вісімкове подання зазначеного аргументу num. Значення найбільшої кількості, яка може бути перетворено, залежить від платформи. Для 32-бітних платформ це зазвичай 4294967295 у десятковому поданні та 37777777777 у вісімковому. Для 64-бітних платформ це зазвичай 9223372036854775807 і 77777777777777777777777 відповідно.

Список параметрів

num Десятичне значення для перетворення

Значення, що повертаються

Восьмеричне рядкове представлення num
