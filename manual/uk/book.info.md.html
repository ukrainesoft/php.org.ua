  • assert_options — Встановлення та отримання налаштувань механізму перевірки тверджень
  • assert — Перевіряє, чи є затвердження false
  • cli_get_process_title - Повертає заголовок поточного процесу
  • cli_set_process_title - Встановлює заголовок процесу
  • dl — Завантажує модуль PHP під час виконання
  • extension_loaded — Визначає, чи завантажений модуль
  • gc_collect_cycles - Примусовий запуск збирача сміття
  • gc_disable — Вимикає збирач циклічних посилань
  • gc_enable — Включає збирач циклічних посилань
  • gc_enabled — Повертає поточний стан збирача циклічних посилань
  • gc_mem_caches - Звільняє пам'ять, яку використовує менеджер пам'яті Zend Engine
  • gc_status — Повертає інформацію про поточному статусі збирача сміття
  • get_cfg_var — Витягує значення налаштування конфігурації PHP
  • get_current_user - Отримує ім'я власника поточного скрипта PHP
  • get_defined_constants - Повертає асоціативний масив з іменами всіх констант та їх значень
  • get_extension_funcs - Повертає масив імен функцій модуля
  • get_include_path — Отримання поточного значення конфігураційної установки include_path
  • get_included_files - Повертає масив імен увімкнених у скрипт файлів
  • get_loaded_extensions - Повертає масив імен усіх скомпілованих та завантажених модулів
  • get_magic_quotes_gpc - Отримання поточного значення налаштування конфігурації magic_quotes_gpc
  • get_magic_quotes_runtime — Отримання поточного значення конфігураційної установки magic_quotes_runtime
  • get_required_files - Псевдонім get_included_files
  • get_resources — Повертає активні ресурси
  • getenv — Отримання значення змінної оточення
  • getlastmod — Отримує час останньої модифікації сторінки
  • getmygid — Отримати GID власника скрипта PHP
  • getmyinode — Отримує значення inode поточного скрипту
  • getmypid — Отримання ID процесу PHP
  • getmyuid — Отримання власника UID скрипта PHP
  • getopt — Отримує настройки зі списку аргументів командного рядка
  • getrusage — Отримує інформацію про використання поточного ресурсу
  • ini_alter - Псевдонім ini_set
  • ini_get_all - Отримує все налаштування конфігурації
  • ini_get — Отримує значення налаштування конфігурації
  • ini_restore - Відновлює значення налаштування конфігурації
  • ini_set — Встановлює значення налаштування конфігурації
  • memory_get_peak_usage - Повертає пікове значення об'єму пам'яті, виділене PHP
  • memory_get_usage - Повертає кількість пам'яті, виділена для PHP
  • php_ini_loaded_file - Отримати шлях до завантаженого файлу php.ini
  • php_ini_scanned_files - Повертає список .ini-файлів, знайдених у додатковій ini-директорії
  • php_sapi_name - Повертає тип інтерфейсу між веб-сервером та PHP
  • php_uname — Повертає інформацію про операційній системі, на якій запущено PHP
  • phpcredits — Виводить список розробників PHP
  • phpinfo — Виводить інформацію про поточну конфігурації PHP
  • phpversion — Отримує поточну версію PHP
  • putenv — Встановлює значення змінного середовища
  • restore_include_path - Відновлює початкове значення конфігурації include_path
  • set_include_path - Встановлює налаштування конфігурації include_path
  • set_time_limit — Обмеження часу виконання скрипту
  • sys_get_temp_dir - Повертає шлях до директорії тимчасових файлів
  • version_compare - Порівнює два "стандартизовані" рядки з номером версії PHP
  • zend_thread_id - Повертає унікальний ідентифікатор поточного потоку виконання
  • zend_version — Отримує версію движка Zend
