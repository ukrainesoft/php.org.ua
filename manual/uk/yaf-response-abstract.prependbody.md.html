Yaf_Response_Abstract::prependBody

(Yaf >=1.0.0)

Yaf_Response_Abstract::prependBody — Призначення prependBody

Опис

public Yaf_Response_Abstract::prependBody(string $content, string $key = ?): bool

Додає вміст до існуючого блоку вмісту

Список параметрів

body Рядок вмісту

key Ключ вмісту, ви можете встановити вміст із ключем, якщо ви не вкажете, то використовуватиметься Yaf_Response_Abstract::DEFAULT_BODY

Примітка:

Параметр додано з 2.2.0

Значення, що повертаються

bool

Приклади

Приклад #1 Приклад використання Yaf_Response_Abstract::prependBody()

<?php$response = new Yaf_Response_Http();$response->setBody("Світ")->prependBody("Привіт ,");echo $response;?>
