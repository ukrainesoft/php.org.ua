POSIX Функції

Також може бути корисним розділ "Функції управління процесами ".

Зміст

  • posix_access — Визначає доступність файлу
  • posix_ctermid — Повертає шлях керуючого терміналу
  • posix_errno - Псевдонім posix_get_last_error
  • posix_get_last_error - Повертає номер помилки, що сталася в останній функції, що завершилася невдачею
  • posix_getcwd — Повертає шлях поточної директорії
  • posix_getegid — Повертає ефективний ідентифікатор групи поточного процесу EGID
  • posix_geteuid — Повертає ефективний ідентифікатор користувача поточного процесу EUID
  • posix_getgid — Повертає дійсний ID групи поточного процесу GID
  • posix_getgrgid — Повертає інформацію про групу з її ID
  • posix_getgrnam — Повертає інформацію про групу, використовуючи її ім'я
  • posix_getgroups — Повертає список груп для поточного процесу
  • posix_getlogin — Повертає логін власника процесу
  • posix_getpgid — Повертає ID групи поточного процесу для менеджера завдань
  • posix_getpgrp — Повертає ідентифікатор групи поточного процесу
  • posix_getpid — Повертає ідентифікатор поточного процесу
  • posix_getppid — Повертає ідентифікатор батьківського процесу
  • posix_getpwnam — Повертає інформацію про користувача на його ім'я
  • posix_getpwuid — Повертає інформацію про користувача, використовуючи його ID
  • posix_getrlimit — Повертає інформацію про обмеження системних ресурсів
  • posix_getsid — Повертає поточний SID процесу
  • posix_getuid — Повертає фактичний ідентифікатор користувача поточного процесу UID
  • posix_initgroups — Визначає рівень доступу для групи
  • posix_isatty — Визначає чи є файловий дескриптор інтерактивним терміналом
  • posix_kill — Надсилає сигнал відповідного процесу
  • posix_mkfifo — Створює спеціальний fifo файл (іменований канал-pipe)
  • posix_mknod — Створює спеціальний або звичайний файл (POSIX.1)
  • posix_setegid — Встановлює ефективний ідентифікатор групи для поточного процесу EGID
  • posix_seteuid — Встановлює ефективний ідентифікатор користувача для поточного процесу EUID
  • posix_setgid — Встановлює ідентифікатор групи для поточного процесу GID
  • posix_setpgid — Встановлює ідентифікатор групи процесу для менеджера завдань
  • posix_setrlimit — Встановлює межі системних ресурсів
  • posix_setsid — Робить поточний процес лідером сесії
  • posix_setuid — Встановлює UID поточного процесу
  • posix_strerror — Повертає системне повідомлення про помилку, ґрунтуючись на отриманому номері помилки
  • posix_times — Повертає інформацію про використання процесорного часу
  • posix_ttyname — Визначає ім'я термінального пристрою
  • posix_uname — Повертає інформацію про системі
