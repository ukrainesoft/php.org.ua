Функції FTP

Зміст

  • ftp_alloc - Резервує місце на диску для закачуваного файлу
  • ftp_append — Додає вміст файлу на кінець іншого файлу на FTP-сервері
  • ftp_cdup — Переходить до батьківської директорію
  • ftp_chdir — Змінює поточну директорію на FTP-сервері
  • ftp_chmod — Встановлює права доступу файлу
  • ftp_close — Закриває з'єднання з FTP-сервером
  • ftp_connect — встановлення з'єднання з FTP-сервером
  • ftp_delete — Видалення файлу на FTP-сервері
  • ftp_exec — Вимагає виконання команди на FTP-сервері
  • ftp_fget — Завантажує файл із FTP-сервера та зберігає його у попередньо відкритому файлі
  • ftp_fput — Завантажує попередньо відкритий файл на FTP-сервер
  • ftp_get_option — Отримує поточні параметри FTP-з'єднання
  • ftp_get — Завантажує файл із FTP-сервера
  • ftp_login — Виконує вхід на сервер FTP
  • ftp_mdtm — Повертає час останнього модифікації файлу
  • ftp_mkdir — Створює директорію
  • ftp_mlsd — Повертає список файлів у заданої директорії
  • ftp_nb_continue — Продовжує асинхронну операцію
  • ftp_nb_fget — Завантажує файл з FTP-сервера в асинхронному режимі і зберігає його попередньо відкритому файлі
  • ftp_nb_fput — Завантажує попередньо відкритий файл на FTP-сервер в асинхронному режимі
  • ftp_nb_get — Завантажує файл з FTP-сервера в асинхронному режимі та зберігає його у локальний файл
  • ftp_nb_put — Завантажує файл на FTP-сервер в асинхронному режимі
  • ftp_nlist — Повертає список файлів у заданої директорії
  • ftp_pasv — Вмикає або вимикає пасивний режим
  • ftp_put — Завантажує файл на сервер FTP
  • ftp_pwd — Повертає ім'я поточної директорії
  • ftp_quit - Псевдонім ftp_close
  • ftp_raw — Надсилає довільну команду FTP-серверу
  • ftp_rawlist – Повертає докладний список файлів у заданій директорії
  • ftp_rename — Перейменування файлу або директорію на FTP-сервері
  • ftp_rmdir — Видаляє каталог
  • ftp_set_option — Встановлює параметри з'єднання з FTP-сервером
  • ftp_site — Надсилає серверу команду SITE
  • ftp_size — Повертає розмір вказаного файлу
  • ftp_ssl_connect — Встановлює з'єднання з FTP-сервером через SSL
  • ftp_systype — Повертає тип операційної системи FTP-сервера
