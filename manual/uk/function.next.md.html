next

(PHP 4, PHP 5, PHP 7, PHP 8)

next - Переміщує покажчик масиву вперед на один елемент

Опис

next(array|object &$array): mixed

next() веде себе подібно current(), але з однією відзнакою. Перед тим, як повернути значення елемента масиву, ця функція пересуває його внутрішній покажчик однією позицію вперед. Іншими словами, вона повертає наступний елемент масиву та зсуває його внутрішній покажчик однією позицію.

Список параметрів

array Масив (array), що змінюється цією функцією.

Значення, що повертаються

Повертає значення елемента масиву, що знаходиться на позиції, наступній за позицією внутрішнього покажчика або false, якщо досягнуто кінця масиву.

Увага

Ця функція може повертати як логічне значення false, так і значення не типу boolean, яке наводиться до false. Більше Детальну інформацію див. у розділі Булев тип. Використовуйте оператор === для перевірки значення, повертається цією функцією.

Список змін

Версія Опис
8.1.0 Виклик функції в об'єкті (object) оголошено застарілим. Або спочатку використовуйте для об'єкта (object) функцію get_mangled_object_vars(), або використовуйте ArrayIterator.

Приклади

Приклад #1 Приклад використання next() та пов'язаних функцій

<?php$transport = array('foot', 'bike', 'car', 'plane');$mode =current($transport); // $mode = 'foot';$mode = next($transport); // $mode = 'bike';$mode = next($transport); // $mode = 'car';$mode = prev($transport); // $mode = 'bike';$mode = end($transport); // $mode = 'plane';?>

Примітки

Примітка: Кінець масиву не відрізняється від елемента масиву типу bool зі значенням false. Для коректного обходу масиву, який може містити false, використовуйте функцію foreach(). Якщо Ви хочете використовувати функцію next() і при цьому коректно відстежувати кінець масиву, перевіряйте, що функція key() повертає null.

Дивіться також

  • current() - Повертає поточний елемент масиву
  • end() - Встановлює внутрішній покажчик масиву на його останній елемент
  • prev() - Пересуває внутрішній покажчик масиву на одну позицію тому
  • reset() - Встановлює внутрішній покажчик масиву на його перший елемент
  • each() - Повертає поточну пару ключ/значення з масиву та зміщує його покажчик
