# Swoole\Buffer::write

(PECL swoole \>u003d 1.9.0)

Swoole\Buffer::write — Записує дані до буфера пам'яті. Пам'ять,
виділена для буфера, не буде змінена

### Опис

public **Swoole\Buffer::write**(int `$offset`, string `$data`): void

### Список параметрів

`offset`
Зміщення.

`data`
Дані для запису буфер пам'яті.
