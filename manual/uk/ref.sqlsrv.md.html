Функції SQLSRV

Зміст

  • sqlsrv_begin_transaction - Розпочинає транзакцію бази даних
  • sqlsrv_cancel — скасовує оператор
  • sqlsrv_client_info - Повертає інформацію про клієнта та зазначене підключення
  • sqlsrv_close — Закриває відкрите з'єднання та звільняє ресурси, пов'язані з цим з'єднанням
  • sqlsrv_commit — Фіксує транзакцію, розпочату за допомогою sqlsrv_begin_transaction
  • sqlsrv_configure — Змінює конфігурацію обробки помилок драйвера та ведення журналу
  • sqlsrv_connect — Відкриває з'єднання з базою даних Microsoft SQL Server
  • sqlsrv_errors — Повертає інформацію про помилку та попередження останньої виконаної операції SQLSRV
  • sqlsrv_execute — Виконує запит, підготовлений за допомогою sqlsrv_prepare
  • sqlsrv_fetch_array - Повертає рядок як масив
  • sqlsrv_fetch_object — Отримує наступний рядок даних у наборі результатів як об'єкт
  • sqlsrv_fetch — Робить наступний рядок у наборі результатів доступної для читання
  • sqlsrv_field_metadata - Отримує метадані для полів оператора, підготовленого за допомогою sqlsrv_prepare або sqlsrv_query
  • sqlsrv_free_stmt — Визволяє все ресурси для вказаного оператора
  • sqlsrv_get_config — Повертає значення вказаного параметра конфігурації
  • sqlsrv_get_field — Отримує дані поля з поточного вибраного рядка
  • sqlsrv_has_rows - Вказує, є у зазначеного оператора рядка
  • sqlsrv_next_result - Робить активним наступний результат вказаного оператора
  • sqlsrv_num_fields — Витягує кількість полів (стовпців) оператора
  • sqlsrv_num_rows — Отримує кількість рядків у наборі результатів
  • sqlsrv_prepare — Підготовка запит до виконання
  • sqlsrv_query — Підготовка та виконує запит
  • sqlsrv_rollback - Відкочує транзакцію, розпочату sqlsrv_begin_transaction
  • sqlsrv_rows_affected - Повертає кількість рядків, змінених останнім виконаним запитом INSERT, UPDATE або DELETE
  • sqlsrv_send_stream_data - Надсилає дані з потоків параметрів на сервер
  • sqlsrv_server_info - Повертає інформацію про сервер
