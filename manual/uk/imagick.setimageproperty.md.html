Imagick::setImageProperty

(PECL imagick 2, PECL imagick 3)

Imagick::setImageProperty — Встановлює властивість зображення

Опис

public Imagick::setImageProperty(string $name, string $value): bool

Встановлює іменовану якість зображення. Цей метод доступний, якщо Imagick був скомпільований із версією ImageMagick 6.3.2 або старшим.

Список параметрів

name

value

Значення, що повертаються

У разі успішної роботи повертає true.
