Клас PharData

(PHP 5 = 5.3.0, PHP 7, PHP 8, PECL phar = 2.0.0)

Вступ

Клас PharData надає високорівневий інтерфейс доступу та створення нездійснюваних tar- та zip-архівів. Оскільки архіви цих типів не містять заглушку і не можуть бути виконані модулем Phar, є можливість створювати та обробляти звичайні zip- та tar-файли, використовуючи клас PharData, навіть якщо параметр phar.readonly у php.ini дорівнює 1.

Зміст

  • PharData::addEmptyDir — Додати порожню директорію в tar/zip-архів
  • PharData::addFile — Додати наявні файли до tar/zip-архіву
  • PharData::addFromString — Створити файл із заданим вмістом у tar/zip-архіві
  • PharData::buildFromDirectory - Створює tar/zip-архів із файлів у директорії
  • PharData::buildFromIterator - Створення tar/zip-архіву за допомогою ітератора
  • PharData::compress - Стискає весь архів tar/zip, використовуючи стиснення Gzip або Bzip2
  • PharData::compressFiles — Стиснути все файли у поточному tar/zip-архіві
  • PharData::__construct - Конструктор об'єкта PharData
  • PharData::convertToData - Конвертація phar-архіву в tar/zip-архів
  • PharData::convertToExecutable - Конвертація tar/zip-архіву з даними в phar-архів, що запускається
  • PharData::copy — Скопіювати файл із tar/zip-архіву в новий файл усередині нього ж
  • PharData::decompress — Розпакувати весь Phar-архів
  • PharData::decompressFiles - Розпакувати всі файли у поточному zip-архіві
  • PharData::delMetadata — Видалити глобальні метадані для zip-архіву
  • PharData::delete — Видалити файл із tar/zip-архіву
  • PharData::__destruct - Знищує нездійснюваний об'єкт архіву tar або zip
  • PharData::extractTo — Вийняти вміст tar/zip-архіву в директорію
  • PharData::isWritable — Перевірити, чи можна модифікувати tar/zip-архів
  • PharData::offsetSet — Зміна вмісту файлу
  • PharData::offsetUnset — Видалити файл із tar/zip-архіву
  • PharData::setAlias — Функція заглушка (Phar::setAlias не можна використовувати для PharData)
  • PharData::setDefaultStub — Функція заглушка (Phar::setDefaultStub не можна використовувати для PharData)
  • PharData::setMetadata — Встановити метадані phar-архіву
  • PharData::setSignatureAlgorithm — Встановити алгоритм підписання phar-архіву та застосування його
  • PharData::setStub — Функція заглушка (Phar::setStub не можна використовувати для PharData)
