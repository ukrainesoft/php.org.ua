#ctype_xdigit

(PHP 4 \>u003d 4.0.4, PHP 5, PHP 7, PHP 8)

ctype_xdigit — Перевірка наявності шістнадцяткових цифр

### Опис

**ctype_xdigit**([mixed](language.types.declarations.md#language.types.declarations.mixed)
`$text`): bool

Перевіряє, чи всі символи в рядку (string) `text`
шістнадцятковими 'цифрами'.

### Список параметрів

`text`
Перевірений рядок.

> **Примітка**:
>
> Якщо передано ціле число (int) у діапазоні між -128 та 255
> включно, воно буде оброблено як ASCII-код одного символу (до
> негативним значенням буде додано 256 для можливості
> представлення символів із розширеного діапазону ASCII). Будь-яке інше
> ціле число буде оброблено як рядок, що містить десяткові цифри
> цього числа.

**Увага**
Починаючи з PHP 8.1.0, передача нерядкових аргументів застаріла. В майбутньому
аргумент буде інтерпретуватися як рядок замість коду ASCII. В
Залежно від передбачуваної поведінки аргумент повинен бути приведений до
рядку (string) або має бути зроблений явний виклик функції
[chr()](function.chr.md).

### Значення, що повертаються

Повертає **`true`**, якщо кожен символ рядка `text` є
шістнадцятковою 'цифрою', тобто десятковою цифрою або символом з
діапазону `[A-Fa-f]`, і **`false`** інакше. При виклику з
порожнім рядком результатом завжди буде **`false`**.

### Приклади

**Приклад #1 Приклад використання **ctype_xdigit()****

` <?php$strings u003d array('AB10BC99', 'AR1012', 'ab12bc99');foreach ($strings as $testcase) {    if (ctype_xdigit($testcase)) {        echo "Строка $testcase состоит из шестнадцатеричных цифр.
";    }}else {         echo "Рядок$testcase складається не тільки із шістнадцяткових цифр.
";    }}?> `

Результат виконання цього прикладу:

Рядок AB10BC99 складається з шістнадцяткових цифр.
Рядок AR1012 складається не лише з шістнадцяткових цифр.
Рядок ab12bc99 складається з шістнадцяткових цифр.
