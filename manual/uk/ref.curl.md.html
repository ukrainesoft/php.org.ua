Функції cURL

Зміст

  • curl_close - Завершує сеанс cURL
  • curl_copy_handle - Копіює дескриптор cURL разом із усіма його налаштуваннями
  • curl_errno — Повертає код останнього помилки
  • curl_error — Повертає рядок з описом останньої помилки поточного сеансу
  • curl_escape — кодує заданий рядок як URL
  • curl_exec — Виконує запит cURL
  • curl_file_create — Створює об'єкт CURLFile

  • curl_init – Ініціалізує сеанс cURL
  • curl_multi_add_handle - Додає звичайний cURL-дескриптор до набору cURL-дескрипторів
  • curl_multi_close - Закриває набір CURL-дескрипторів
  • curl_multi_errno - Повертає код останньої помилки множинного curl
  • curl_multi_exec - Запускає приєднання поточного дескриптора cURL
  • curl_multi_getcontent - Повертає результат операції, якщо було встановлено опцію CURLOPT_RETURNTRANSFER
  • curl_multi_info_read - Повертає інформацію про поточні операції
  • curl_multi_init — Створює набір CURL-дескрипторів

  • curl_multi_setopt — Встановити опції для множинного дескриптора cURL
  • curl_multi_strerror - Повертає рядок, що описує помилку
  • curl_pause — Зупинити та відновити з'єднання
  • curl_reset — Скинути всі налаштування обробника сесії libcurl
  • curl_setopt_array - Встановлює кілька параметрів для сеансу cURL
  • curl_setopt — Встановлює параметр для сеансу CURL
  • curl_share_close — Закрити оброблюваний обробник cURL
  • curl_share_errno — Повертає код останньої помилки розділяється обробника curl
  • curl_share_init - Ініціалізація оброблюваного обробника cURL
  • curl_share_setopt — Встановити опції оброблюваного cURL
  • curl_share_strerror - Повертає опис для заданого коду помилки
  • curl_strerror — Отримати текстове опис для коду помилки
  • curl_unescape - Декодує закодований URL-рядок
  • curl_version - Повертає версію cURL
