# cos

(PHP 4, PHP 5, PHP 7, PHP 8)

cos — Косінус

### Опис

**cos**(float `$num`): float

**cos()** повертає косинус параметра `num`. Параметр `num` задається в
радіанах.

### Список параметрів

`num`
Кут у радіанах

### Значення, що повертаються

Косинус кута `num`

### Приклади

**Приклад #1 Приклад використання **cos()****

`<?phpecho cos(M_PI); // -1?> `
