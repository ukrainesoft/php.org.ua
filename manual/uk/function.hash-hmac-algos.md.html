#hash_hmac_algos

(PHP 7 >= 7.2.0, PHP 8)

hash_hmac_algos — Повертає список зареєстрованих алгоритмів хешування, що застосовуються для hash_hmac

Опис

hash_hmac_algos(): array

Список параметрів

Ця функція не має параметрів.

Значення, що повертаються

Повертає індексований масив зі списком підтримуваних функцією hash_hmac() алгоритмів хешування.

Приклади

Приклад #1 Приклад використання hash_hmac_algos()

<?phpprint_r(hash_hmac_algos());

Примітки

Примітка:

До PHP 7.2.0 єдиним варіантом отримати список підтримуваних алгоритмів було викликати функцію hash_algos(), яка також повертала алгоритми, що не підтримуються hash_hmac().
