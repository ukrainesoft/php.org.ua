Передбачені класи

У цьому розділі перераховуються стандартні визначені класи. Різноманітні модулі визначають інші класи, які описані в відповідної довідкової інформації.

Стандартні певні класи

Ці класи визначені разом зі стандартним набором функцій, що йдуть збиранням PHP.

Directory Створюється функцією dir().

stdClass Створюється наведенням типу до об'єкту.

__PHP_Incomplete_Class Можливо, створюється функцією unserialize().

Exception

ErrorException

php_user_filter

Closure Обумовлений остаточний клас Closure, використовується для внутрішньої реалізації анонімних функцій.

Generator Обумовлений остаточний клас Generator, використовується для представлення генераторів.

DivisionByZeroError

Спеціальні класи

Наступні ідентифікатори не можуть використовуватись як ім'я класу, оскільки вони мають спеціальне призначення.

self Поточний клас.

static Поточний клас під час виконання.

parent Батьківський клас.
