Клас ZipArchive

(PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.1.0)

Вступ

Файловий архів, стислий Zip.

Огляд класів

Властивості

lastId Значення індексу останнього доданого запису (файл або каталог). Доступно з PHP 8.0.0 та PECL zip 1.18.0.

status Заголовок Zip-архіву. Доступно для закритого архіву, починаючи з PHP 8.0.0 та PECL ZIP 1.18.0.

statusSys Системний статус zip-архіву. Доступно для закритого архіву, починаючи з PHP 8.0.0 та PECL zip 1.18.0.

numFiles Кількість файлів в архіві

filename Ім'я файлу у файловій системі

comment Коментар до архіву

Зміст

  • ZipArchive::addEmptyDir - Додає нову директорію
  • ZipArchive::addFile — Додає до ZIP-архів файл за вказаним шляхом
  • ZipArchive::addFromString - Додає файл до ZIP-архіву, використовуючи його вміст
  • ZipArchive::addGlob — Додати файли з директорії відповідно до шаблону
  • ZipArchive::addPattern — Додати файли з директорії відповідно до шаблону регулярного виразу PCRE
  • ZipArchive::clearError — Видаляє повідомлення про помилку статусу, системні та/або повідомлення модуля zip
  • ZipArchive::close - Закриває активний архів (відкритий або знову створений)
  • ZipArchive::count — Підраховує кількість файлів в архіві
  • ZipArchive::deleteIndex — Видаляє елемент в архіві, використовуючи його індекс
  • ZipArchive::deleteName — Видаляє елемент в архіві, використовуючи його ім'я
  • ZipArchive::extractTo - Витягує вміст архіву
  • ZipArchive::getArchiveComment - Повертає коментар ZIP-архіву
  • ZipArchive::getCommentIndex - Повертає коментар елемента, використовуючи його індекс
  • ZipArchive::getCommentName - Повертає коментар елемента, використовуючи його ім'я
  • ZipArchive::getExternalAttributesIndex — Витягти зовнішні атрибути запису за його індексом
  • ZipArchive::getExternalAttributesName — Витягти зовнішні атрибути запису на її ім'я
  • ZipArchive::getFromIndex - Повертає вміст елемента за його індексом
  • ZipArchive::getFromName - Повертає вміст елемента на його ім'я
  • ZipArchive::getNameIndex Повертає ім'я елемента за його індексом
  • ZipArchive::getStatusString - Повертають статус повідомлення про помилку, системний та/або zip-статус
  • ZipArchive::getStream — Отримати дескриптор файлу елемента, визначений на ім'я елемента (тільки для читання)
  • ZipArchive::getStreamIndex - Отримує обробник файлу для запису, визначеного його індексом (тільки для читання)
  • ZipArchive::getStreamName - Отримує обробник файлу для запису, визначеного його ім'ям (тільки для читання)
  • ZipArchive::isCompressionMethodSupported — Перевіряє, чи підтримується метод стиснення libzip
  • ZipArchive::isEncryptionMethodSupported — Перевіряє, чи підтримується метод шифрування libzip
  • ZipArchive::locateName - Повертає індекс елемента в архіві
  • ZipArchive::open - Відкриває ZIP-архів
  • ZipArchive::registerCancelCallback — Реєструє callback-функцію для дозволу скасування під час закриття архіву
  • ZipArchive::registerProgressCallback — Реєструє callback-функцію для надання оновлень при закриття архіву
  • ZipArchive::renameIndex - Перейменовує елемент за його індексом
  • ZipArchive::renameName - Перейменовує елемент на його ім'я
  • ZipArchive::replaceFile — Замінює файл у ZIP-архіві вказаним шляхом
  • ZipArchive::setArchiveComment - Встановлює коментар до ZIP-архіву
  • ZipArchive::setCommentIndex - Встановлює коментар до елемента за його індексом
  • ZipArchive::setCommentName - Встановлює коментар до елемента, заданого на ім'я
  • ZipArchive::setCompressionIndex — Встановити метод стиснення запису, заданого його індексом
  • ZipArchive::setCompressionName — Встановити метод стиснення запису, заданого на ім'я
  • ZipArchive::setEncryptionIndex — Встановити метод шифрування запису за його індексом
  • ZipArchive::setEncryptionName - Встановити метод шифрування запису на його ім'я
  • ZipArchive::setExternalAttributesIndex — Встановити зовнішні атрибути запису за його індексом
  • ZipArchive::setExternalAttributesName — Встановлення зовнішніх атрибутів запису, заданого на ім'я
  • ZipArchive::setMtimeIndex - Встановити час модифікації файлу за його індексом
  • ZipArchive::setMtimeName - Встановити час модифікації файлу на його ім'я
  • ZipArchive::setPassword — Встановлення пароля для активного архіву
  • ZipArchive::statIndex - Отримання детальної інформації про елемент за його індексом
  • ZipArchive::statName — Отримання детальної інформації про елемент за його ім'ям
  • ZipArchive::unchangeAll — Скасує всі зміни, зроблені в архіві
  • ZipArchive::unchangeArchive Повертає всі глобальні зміни, зроблені в архіві
  • ZipArchive::unchangeIndex Скасує всі зміни у позиції із заданим індексом
  • ZipArchive::unchangeName — Скасує усі зміни у позиції із заданим ім'ям
