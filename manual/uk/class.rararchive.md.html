Клас RarArchive

(PECL rar >= 2.0.0)

Вступ

Цей клас описує RAR архів, який може складатися з кількох томів (частин) і містити кілька RAR записів (файли, директорії та інші спеціальні об'єкти, такі як символічні посилання).

Об'єкти цього класу можуть бути заповнені перебором на основі записів, що зберігаються у відповідному RAR архіві. Ці записи можуть бути також отримані за допомогою методів RarArchive::getEntry() та RarArchive::getEntries().

Огляд класів

final class RarArchive implements Traversable {

/* Методи */

public close(): bool

public getComment(): string

public getEntries(): array|false

public getEntry(string $entryname): RarEntry|false

public isBroken(): bool

public isSolid(): bool

public static open(string $filename, string $password = NULL, callable $volume_callback = NULL): RarArchive|false

public setAllowBroken(bool $allow_broken): bool

public __toString(): string

}

Зміст

  • RarArchive::close - Закриває RAR архів та звільняє всі ресурси
  • RarArchive::getComment — Отримати текст коментаря з архіву RAR
  • RarArchive::getEntries - Повертає повний список елементів з RAR архіву
  • RarArchive::getEntry — Повертає об'єкт елемента з RAR архіву
  • RarArchive::isBroken — Перевіряє, чи не чи зламаний архів (не завершено)
  • RarArchive::isSolid — Перевірити, чи є чи архів суцільним
  • RarArchive::open - Відкриває архів RAR
  • RarArchive::setAllowBroken Чи відкривати пошкоджені архіви
  • RarArchive::__toString — Отримати текстове уявлення
