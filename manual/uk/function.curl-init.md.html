#curl_init

(PHP 4 >= 4.0.2, PHP 5, PHP 7, PHP 8)

curl_init – Ініціалізує сеанс cURL

Опис

curl_init(?string $url = null): CurlHandle|false

Ініціалізує новий сеанс cURL та повертає дескриптор, який використовується з функціями curl_setopt(), curl_exec() та curl_close().

Список параметрів

url Якщо вказано, опція CURLOPT_URL буде автоматично встановлена в значення цього аргументу. Ви можете вручну встановити цю опцію з за допомогою функції curl_setopt().

Примітка:

Протокол file стає недоступним у cURL, якщо задана опція open_basedir.

Значення, що повертаються

Повертає дескриптор cURL у разі успішного виконання, і false у разі виникнення помилки.

Список змін

Версія Опис
8.0.0 У разі успішного виконання повертає екземпляр CurlHandle; раніше, повертався ресурс (resource).
8.0.0 url тепер припускає значення null.

Приклади

Приклад #1 Ініціалізація нового сеансу cURL та завантаження веб-сторінки

<?php// створення нового ресурсу cURL$ch = curl_init();// установка URL і інших необхідних параметрівcurl_setopt($ch, CURLOPT_URL, "http://www.example.com CURLOPT_HEADER, 0);// завантаження сторінки і видача еї браузеруcurl_exec($ch);// завершення сеансу і звільнення ресурсовcurl_close($ch);?>
