mysqli_free_result

(PHP 5, PHP 7, PHP 8)

mysqli_result::free -- mysqli_result::close -- mysqli_result::free_result -- mysqli_free_result — Звільняє пам'ять, зайняту результатами запиту

Опис

Об'єктно-орієнтований стиль

public mysqli_result::free(): void

public mysqli_result::close(): void

public mysqli_result::free_result(): void

Процедурний стиль

mysqli_free_result(mysqli_result $result): void

Визволяє пам'ять, зайняту результатами запиту.

Список параметрів

result Тільки для процедурного стилю: об'єкт mysqli_result, отриманий за допомогою mysqli_query(), mysqli_store_result(), mysqli_use_result() або mysqli_stmt_get_result().

Значення, що повертаються

Функція не повертає значення після виконання.

Дивіться також

  • mysqli_query() - Виконує запит до бази даних
  • mysqli_stmt_get_result() - Отримує результат із підготовленого запиту як об'єкт mysqli_result
  • mysqli_store_result() - Передає на клієнта результуючий набір останнього запиту
  • mysqli_use_result() - Готує результуючий набір на сервері для використання
