tidy::getConfig

tidy_get_config

(PHP 5, PHP 7, PHP 8, PECL tidy = 0.7.0)

tidy::getConfig -- tidy_get_config -- Отримує поточну конфігурацію Tidy

Опис

Об'єктно-орієнтований стиль

public tidy::getConfig(): array

Процедурний стиль

tidy_get_config(tidy $tidy): array

Отримує список опцій конфігурації, що використовуються у вказаному об'єкті tidy tidy.

Список параметрів

tidy Об'єкт Tidy.

Значення, що повертаються

Повертає масив опцій конфігурації.

Інформацію про кожен параметр можна отримати тут: »http://api.md-tidy.org/#quick-reference.
