(PHP 4 >= 4.0.7, PHP 5, PHP 7, PHP 8)

imagefttext — Нанесення тексту на зображення за допомогою шрифтів FreeType 2

Опис

imagefttext( GdImage $image, float $size, float $angle, int $x, int $y, int $color, string $font_filename, string $text, array $options = [] ): array|false

Примітка:

До PHP 8.0.0 imagefttext() - це розширений варіант imagettftext(), який додатково підтримує extrainfo. Починаючи з PHP 8.0.0, imagettftext() є псевдонімом imagefttext().

Список параметрів

image Об'єкт GdImage, який повертається однією з функцій створення зображень, наприклад, такий як imagecreatetruecolor().

size Розмір шрифту у пунктах.

angle Кут у градусах, 0 градусів означає розташування тексту зліва направо. Позитивні значення означає поворот тексту проти годинникової стрілки. Наприклад, текст повернутий на 90 градусів потрібно буде читати знизу вгору.

x Координати x та y визначають відправну точку для першого символу текст (конкретно, лівий нижній кут символу). Тут є відмінність від функції imagestring(), в якій x та y визначають лівий верхній кут першого символу. Наприклад, "верхній лівий" має координати 0,0.

y y-координат. Це позиція базової лінії шрифту, у загальному випадку вона не збігається із нижчою точкою у символі.

col Індекс потрібного кольору тексту дивіться imagecolorexact().

font_filename Шлях до TrueType шрифту, який потрібно використовувати.

Залежно від версії GD бібліотеки якщо font_filename не починається з /, то в кінець назви файлу буде додано розширення .ttf, і бібліотека намагатиметься знайти цей файл за адресою, визначеному в установках бібліотеки.

У більшості випадків розміщення файлів шрифтів у директорії скрипту вирішує подібні проблеми увімкнення файлів.

<?php// Задание переменной окружения для GDputenv('GDFONTPATH=' . realpath('.'));// Имя шрифта для использования (обратите внимание на отсутствие расширения .ttf)$font = 'SomeFont';?>

text Текст для вставлення зображення.

options

Ключ Тип Значення
linespacing float Визначає малювання нижнього підкреслення

Можливі значення масиву options

Значення, що повертаються

Ця функція повертає масив, який визначає чотири точки рамки тексту. Текст усередині цих кордонів починається з лівого нижнього кута і повертається проти годинникової стрілки:
0 нижня ліва x-координата
1 нижня ліва y-координата
2 нижня права x-координата
3 нижня права y-координата
4 верхня права x-координата
5 верхня права y-координата
6 верхня ліва x-координата
7 верхня ліва y-координата

У разі виникнення помилки повертає false.

Список змін

Версія Опис
8.0.0 image тепер чекає екземпляр GdImage; раніше очікували ресурс (resource).
