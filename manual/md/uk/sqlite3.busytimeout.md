# SQLite3::busyTimeout

(PHP 5 \>u003d 5.3.3, PHP 7, PHP 8)

SQLite3::busyTimeout — Встановити обробник "зайнято" на з'єднання

### Опис

public **SQLite3::busyTimeout**(int `$milliseconds`): bool

Встановлює обробник, який спатиме доти, доки БД не
заблокується або поки не закінчиться час очікування.

### Список параметрів

`milliseconds`
Мілісекунди. Установка дорівнює 0 або негативному значенню призведе до
відключення вже встановленого обробника.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у
у разі виникнення помилки.
