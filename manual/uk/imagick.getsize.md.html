Imagick::getSize

(PECL imagick 2, PECL imagick 3)

Imagick::getSize — Повертає розмір, пов'язаний із об'єктом Imagick

Опис

public Imagick::getSize(): array

Отримує розмір пікселів об'єкта Imagick, попередньо заданий з за допомогою функції Imagick::setSize().

Примітка:

Цей метод лише повертає розмір, встановлений Imagick::setSize(). Якщо вам потрібно отримати реальні розміри зображення, використовуйте функції Imagick::getImageWidth() та Imagick::getImageHeight().

Список параметрів

Ця функція не має параметрів.

Значення, що повертаються

Повертає розмір, пов'язаний із об'єктом Imagick у вигляді масиву з ключами "columns" (стовпці) та "rows" (рядки).

Приклади

Приклад #1 Отримання розміру вихідного зображення RGB, встановлені в 200x400, після масштабування до 400x800

<?php//Встановлюємо розмір і завантажуємо зображення$img = new Imagick();$img->setSize(200, 400);$img->readImage("image.rgb");$img->scaleImage(400 , 800);$size = $img->getSize();print_r($size);echo $img->getImageWidth()."x".$img->getImageHeight();?>

Результат виконання цього прикладу:

Array ( [columns] => 200 [rows] => 400 ) 400x800
