## Список інших зарезервованих слів

Наступні слова не можна використовувати в іменах класів, інтерфейсів та
трейтів. Також їх заборонено використовувати в назві просторів імен.

| | | | |
|----------------------|--------------------|----- --------------|-------------------|
| int | float | bool | string |
| true | false | null | void (з PHP 7.1)
| iterable (з PHP 7.1) об'єкт (з PHP 7.2) | mixed (з PHP 8.0) never (з PHP 8.1)

**Зарезервовані слова**

Наступний перелік містить м'яко зарезервовані слова. Вони все ще
можуть використовуватися як імена класів, інтерфейсів і трейтів (а
також у просторах імен), але використовувати їх вкрай не рекомендується,
оскільки вони можуть стати повноцінно зарезервованими у майбутньому.

| | | | |
|------|----------|---------|-----|
| enum | resource | numeric | |
