# tanh

(PHP 4 \>u003d 4.1.0, PHP 5, PHP 7, PHP 8)

tanh - Гіперболічний тангенс

### Опис

**tanh**(float `$num`): float

Повертає гіперболічний тангенс `num`, який визначається як
`sinh(arg)/cosh(arg)`.

### Список параметрів

`num`
Вхідне значення

### Значення, що повертаються

Гіперболічний тангенс `num`
