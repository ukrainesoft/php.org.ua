bcdiv

(PHP 4, PHP 5, PHP 7, PHP 8)

bcdiv - Операція поділу для чисел довільної точності

Опис

bcdiv(string $num1, string $num2, ?int $scale = null): string

Поділяє num1 на num2.

Список параметрів

num1 Подільне у вигляді рядка.

num2 Дільник у вигляді рядка.

scale Цей необов'язковий параметр використовується для встановлення кількості знаків після десяткового роздільника у результаті. Якщо не поставлено, то, за замовчуванням, буде використано значення, задане глобально за допомогою bcscale(), або 0.

Значення, що повертаються

Повертає результат поділу у вигляді рядка або null, якщо divisor дорівнює 0.

Список змін

Версія Опис
8.0.0 scale тепер припускає значення null.

Приклади

Приклад #1 Приклад використання bcdiv()

<?phpecho bcdiv('105', '6.55957', 3); // 16.007?>
