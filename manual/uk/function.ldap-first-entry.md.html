#ldap_first_entry

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_first_entry — Повернути перший ідентифікатор результату

Опис

ldap_first_entry(LDAP\Connection $ldap, LDAP\Result $result): LDAP\ResultEntry|false

Повертає ідентифікатор для першого запису в результаті. Цей ідентифікатор потім використовується у функції ldap_next_entry(), для послідовного отримання записів із результату.

Записи в LDAP результаті зчитуються послідовно використовуючи функції ldap_first_entry() та ldap_next_entry().

Список параметрів

ldap Примірник LDAP\Connection, що повертається функцією ldap_connect().

result Примірник LDAP\Result, що повертається ldap_list() або ldap_search().

Значення, що повертаються

Повертає екземпляр LDAP\ResultEntry або false у разі виникнення помилки.

Список змін

Версія Опис
8.1.0 Параметр ldap тепер очікує на екземпляр LDAP\Connection; раніше очікувався ресурс (resource).
8.1.0 Параметр result тепер чекає на екземпляр LDAP\Result; раніше очікувався ресурс (resource).
8.1.0 Повертає екземпляр LDAP\ResultEntry; раніше повертався ресурс (resource).
