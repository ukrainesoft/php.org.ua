#gnupg_sign

(PECL gnupg >= 0.1)

gnupg_sign — Підписує переданий текст

Опис

gnupg_sign(resource $identifier, string $plaintext): string

Підписує переданий у параметрі plaintext текст ключем, що був раніше встановлений за допомогою gnupg_addsignkey та повертає підписаний текст або підпис, залежно від того, що було встановлено в gnupg_setsignmode.

Список параметрів

identifier Ідентифікатор gnupg, отриманий з gnupg_init() або gnupg.

plaintext Простий текст для підписання.

Значення, що повертаються

У разі успішного виконання функція повертає підписаний текст або підпис. У разі помилки функція повертає false.
