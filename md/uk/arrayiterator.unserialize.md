# ArrayIterator::unserialize

(PHP 5 \>u003d 5.3.0, PHP 7, PHP 8)

ArrayIterator::unserialize — Десеріалізація

### Опис

public **ArrayIterator::unserialize**(string `$data`): void

**Увага**

ознайомлення доступний лише список аргументів.

### Список параметрів
