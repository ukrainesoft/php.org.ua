Зміст

  • com_create_guid - Створення унікального глобального ідентифікатора (GUID)
  • com_event_sink — Зв'язати повідомлення об'єкт COM з об'єктом PHP
  • com_get_active_object - Повернути дескриптор на запущений екземпляр об'єкта COM
  • com_load_typelib — Завантаження Typelib
  • com_message_pump - Обробка повідомлень COM, що прийшли не пізніше timeoutms мілісекунд після неї запуску
  • com_print_typeinfo - Друкує визначення класу PHP для інтерфейсу, що наслідує IDispatch
  • variant_abs — Отримати повне значення варіанта
  • variant_add — Скласти значення двох варіантів
  • variant_and - Побітове І над двома варіантами
  • variant_cast - Перетворення варіанта у новий варіант іншого типу
  • variant_cat - Об'єднання (конкатенація) значень двох варіантів
  • variant_cmp — Порівняти два варіанти
  • variant_date_from_timestamp — Отримати подання дати для варіанта з тимчасової мітки Unix
  • variant_date_to_timestamp — Перетворює варіант типу дата/час у часову мітку Unix
  • variant_div — Отримати результат поділу двох варіантів
  • variant_eqv - Побітова еквівалентність двох варіантів
  • variant_fix - Повернути цілу частину варіанти
  • variant_get_type — Отримати тип варіанти
  • variant_idiv - Перетворення варіантів до цілих з наступним розподілом
  • variant_imp - Побітова імплікація над двома варіантами
  • variant_int - Повернути цілу чисельну частина варіанта
  • variant_mod - Залишок від поділу двох варіантів
  • variant_mul — Збільшує значення двох варіантів
  • variant_neg - Логічне заперечення над варіантом
  • variant_not - Побітове заперечення над варіантом
  • variant_or - Побітове АБО над двома варіантами
  • variant_pow — Зводить один варіант у ступінь, заданий у другому
  • variant_round — Округлює варіант з заданою точністю
  • variant_set_type — Приведення варіанти до іншого типу "за місцем"
  • variant_set - Присвоєння нового значення об'єкту варіанта
  • variant_sub — Віднімає значення правого варіанти з лівого
  • variant_xor - Виключає АБО над двома варіантами
