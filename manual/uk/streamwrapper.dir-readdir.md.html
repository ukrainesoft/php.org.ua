streamWrapper::dir_readdir

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

streamWrapper::dir_readdir — Читання запису з дескриптора директорії

Опис

public streamWrapper::dir_readdir(): string

Цей метод викликається у процесі виконання readdir().

Список параметрів

Ця функція не має параметрів.

Значення, що повертаються

Повинна повертати рядок (string), який представляє ім'я наступного файлу, або false, якщо наступного файлу немає.

Примітка:

Значення, що повертається, буде наведено до типу string.

Помилки

Викликає помилку рівня E_WARNING, якщо виклик до цього методу не удався (наприклад, не реалізований).
