# sodium_crypto_core_ristretto255_scalar_mul

(PHP 8 \>u003d 8.1.0)

sodium_crypto_core_ristretto255_scalar_mul — Збільшує скалярне значення

### Опис

**sodium_crypto_core_ristretto255_scalar_mul**(string `$x`, string
`$y`): string

Помножує скалярне значення. Доступно з libsodium 1.0.18.

**Увага**

На цей час ця функція ще була документована; для
ознайомлення доступний лише список аргументів.

### Список параметрів

`x`
Скаляр, який представляє координату X.

`y`
Скаляр, який представляє координату Y.

### Значення, що повертаються

Повертає випадковий 32-байтовий рядок (string).
