mysqli_result::field_seek

mysqli_field_seek

(PHP 5, PHP 7, PHP 8)

mysqli_result::field_seek -- mysqli_field_seek — Встановити покажчик поля на певне усунення

Опис

Об'єктно-орієнтований стиль

public mysqli_result::field_seek(int $index): bool

Процедурний стиль

mysqli_field_seek(mysqli_result $result, int $index): bool

Встановлює вказівник поля на задане усунення. Наступний виклик mysqli_fetch_field() дозволить отримати інформацію про стовпчик з позицією усунення.

Примітка:

Для пошуку з початку рядка необхідно встановити нульове значення усунення.

Список параметрів

result Тільки для процедурного стилю: об'єкт mysqli_result, отриманий за допомогою mysqli_query(), mysqli_store_result(), mysqli_use_result() або mysqli_stmt_get_result().

index Номер поля. Це значення має бути в наступному діапазоні: від 0 до число полів - 1.

Значення, що повертаються

Повертає true у разі успішного виконання або false у у разі виникнення помилки.
