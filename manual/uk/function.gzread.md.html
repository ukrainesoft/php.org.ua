gzread

(PHP 4, PHP 5, PHP 7, PHP 8)

gzread — Бінарне читання gz-файлу

Опис

gzread(resource $stream, int $length): string|false

gzread() читає (розпаковуючи) до length байт з gz-файлу з переданого покажчика gz-файлу. Читання закінчується при досягненні максимального розміру length розпакованих даних або кінця файлу (EOF)

Список параметрів

stream Вказівник на gz-файл, повернутий після його успішного відкриття, функцією gzopen().

length Кількість байт (після розпакування), які слід прочитати.

Значення, що повертаються

Прочитані дані або false у разі виникнення помилки.

Список змін

Версія Опис
7.4.0 У разі виникнення помилки повертається false; раніше, повертався 0.

Приклади

Приклад #1 Приклад використання gzread()

<?php// помістити вміст gz-файлу в рядок$filename = "/usr/local/something.txt.gz";$zd = gzopen($filename, "r");$contents = gzread($zd 10000); gzclose ($zd);?>
