Фреймворк хеш-кодів HASH

  • Вступ
  • Встановлення та налаштування
  • Вимоги
  • Установка
  • Налаштування під час виконання
  • Типи ресурсів
  • Предвизначені константи
  • HashContext - Клас HashContext
  • HashContext::__construct - Закритий конструктор для заборони безпосереднього створення об'єкта
  • HashContext::__serialize - Серіалізує об'єкт HashContext
  • HashContext::__unserialize - Десеріалізує параметр data в об'єкті HashContext
  • Функції Hash
  • hash_algos — Повертає список зареєстрованих алгоритмів хешування
  • hash_copy — Копіює контекст хешування
  • hash_equals - Порівняння рядків, нечутливе до атак за часом
  • hash_file - Генерація хеш-значення, використовуючи вміст заданого файлу
  • hash_final - Завершує інкрементальне хешування та повертає результат у вигляді хеш-коду
  • hash_hkdf — Формування ключа HKDF для заданих вхідних даних
  • hash_hmac_algos - Повертає список зареєстрованих алгоритмів хешування, які застосовуються для hash_hmac
  • hash_hmac_file - Генерація хеш-коду на основі ключа, використовуючи метод HMAC та вміст отриманого файлу
  • hash_hmac - Генерація хеш-коду на основі ключа, використовуючи метод HMAC
  • hash_init - Ініціалізація інкрементального контексту хешування
  • hash_pbkdf2 — Формування ключа PBKDF2 для заданих вхідних даних
  • hash_update_file - Додає дані з файлу в активний контекст хешування
  • hash_update_stream - Додає дані з відкритого потоку до активного контексту хешування
  • hash_update — Додає дані до активний контекст хешування
  • hash - Генерує хеш-код (підпис повідомлення)
