Інтерфейс Collection

(No version information available, might only be in Git)

Вступ

Collection – це базовий інтерфейс, який покриває функціональність загальну всім структур даних у цій бібліотеці. Він гарантує, що це структури обхідні, счётный і може бути перетворені на JSON за допомогою функції json_encode().

Огляд інтерфейсів

class Ds\Collection implements Countable, IteratorAggregate, JsonSerializable {

/* Методи */

abstract public clear(): void

abstract public copy(): Ds\Collection

abstract public isEmpty(): bool

abstract public toArray(): array

}

Список змін

Версія Опис
PECL DS 1.4.0 Клас Collection тепер реалізує інтерфейс IteratorAggregate замість Traversable. (Ця зміна з'явилася у поліфілі у версії 1.4.1).

Зміст

  • Ds\Collection::clear — Видаляє все значення
  • Ds\Collection::copy — Повертає копію колекції
  • Ds\Collection::isEmpty — Перевіряє, чи порожня колекція
  • Ds\Collection::toArray - Перетворює колекцію в масив (array)
