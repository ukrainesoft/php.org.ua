#imap_mailboxmsginfo

(PHP 4, PHP 5, PHP 7, PHP 8)

imap_mailboxmsginfo — Отримати інформацію про поточну поштову скриньку

Опис

imap_mailboxmsginfo(IMAP\Connection $imap): stdClass

Перевірка статусу поточної поштової скриньки на сервері. Аналогічно imap_status(), але додатково обчислює сумарний розмір усіх листів у ящику, через що працює кілька повільніше.

Список параметрів

imap Примірник IMAP\Connection.

Значення, що повертаються

Повертає об'єкт із наступними полями:
Date дата останньої зміни (поточна дата та час)
Driver драйвер
Mailbox ім'я поштової скриньки
Nmsgs кількість листів
Recent кількість нових
Unread кількість непрочитаних
Deleted кількість віддалених
Size розмір ящика

Властивості поштової скриньки

У разі виникнення помилки повертає false.

Список змін

Версія Опис
8.1.0 Параметр imap тепер чекає на екземпляр IMAP\Connection; раніше очікувався ресурс (resource).
