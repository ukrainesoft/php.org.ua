#gmp_powm

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

gmp_powm - Зводить число в ступінь і виробляє розподіл за модулем

Опис

gmp_powm(GMP|int|string $num, GMP|int|string $exponent, GMP|int|string $modulus): GMP

Обчислює (num зводиться в ступінь exponent) залишок від цілісного розподілу на modulus. Якщо exponent негативний, результат не визначено.

Список параметрів

num Підстава ступеня.

Об'єкт GMP, ціле число (int) або числовий рядок (string).

exponent Позитивний показник ступеня, в який зводиться 'num'.

Об'єкт GMP, ціле число (int) або числовий рядок (string).

modulus Дільник, залишок від цілого розподілу на який буде повернутий.

Об'єкт GMP, ціле число (int) або числовий рядок (string).

Значення, що повертаються

Нове число GMP (результат операції).

Приклади

Приклад #1 Приклад використання gmp_powm()
