Зміст

  • connection_aborted — Перевірити, чи відключений клієнт
  • connection_status — Повертає статус з'єднання в бітах
  • constant — Повертає значення константи

  • exit — Вивести повідомлення та припинити виконання поточного скрипту
  • get_browser — Повідомляє про можливості браузера користувача

  • sapi_windows_vt100_support — Отримати або встановити підтримку VT100 для заданого потоку, пов'язаного з буфером виведення консолі Windows
  • show_source - Псевдонім highlight_file

  • sys_getloadavg — Отримує середню завантаження системи

  • time_sleep_until - Відкладає виконання скрипту до заданого часу
  • uniqid — Створення унікального ID
  • unpack — Розпакувати дані з бінарної рядки
  • usleep — Затримка виконання у мікросекундах
