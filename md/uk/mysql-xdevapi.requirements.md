## Вимоги

Для цього модуля потрібен сервер MySQL 8+ з увімкненим плагіном X (по
замовчуванням).

Необхідні бібліотеки для компіляції цього модуля: Boost (1.53.0 або
вище), OpenSSL та Protobuf.
