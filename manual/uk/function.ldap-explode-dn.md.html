#ldap_explode_dn

(PHP 4, PHP 5, PHP 7, PHP 8)

ldap_explode_dn — Розділити DN на його складові

Опис

ldap_explode_dn(string $dn, int $with_attrib): array|false

Поділяє DN, повернутий функцією ldap_get_dn(), і розбиває його на складові частини. Кожна частина відома як відносне відмінне ім'я або RDN.

Список параметрів

dn Відмінна назва об'єкта LDAP.

with_attrib Використовується для запиту, якщо RDNs повернені лише зі значеннями або також зі своїми атрибутами. Для отримання RDNs з атрибутами (тобто форматі attribute = value) встановіть параметр with_attrib в 0 і для отримання лише значень встановіть його в 1.

Значення, що повертаються

Повертає масив всіх компонентів DN або false у разі виникнення помилки. Перший елемент у цьому масиві має ключ count, і він є числом повернутих значень. Наступні елементи – це компоненти DN із цифровими індексами.
