Клас php_user_filter

(PHP 5, PHP 7, PHP 8)

Вступ

Нащадки цього класу передаються у функцію stream_filter_register(). Зверніть увагу, що метод __construct не викликається; натомість для ініціалізації слід використовувати php_user_filter::onCreate().

Огляд класів

class php_user_filter {

/* Властивості */

public string $filtername = "";

public mixed $params = "";

public ?resource $stream = null;

/* Методи */

public filter( resource $in, resource $out, int &$consumed, bool $closing ): int

public onClose(): void

public onCreate(): bool

}

Властивості

filtername Ім'я фільтра, зареєстрованого функцією stream_filter_append().

params

stream

Зміст

  • php_user_filter::filter — Викликається, як тільки застосовується фільтр
  • php_user_filter::onClose - Викликається під час закриття фільтра
  • php_user_filter::onCreate - Викликається під час створення об'єкта фільтра
