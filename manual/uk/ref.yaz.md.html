Функції YAZ

Зміст

  • yaz_addinfo — Повертає додаткову інформацію інформацію у разі виникнення помилки
  • yaz_ccl_conf - Конфігурує CCL-парсер
  • yaz_ccl_parse — Викликає парсер CCL
  • yaz_close — Закриває з'єднання YAZ
  • yaz_connect — Підготовка з'єднання із сервером Z39.50
  • yaz_database — Визначає бази даних у сеансі
  • yaz_element — Вказує ім'я набору елементів для пошуку
  • yaz_errno — Повертає номер помилки

  • yaz_hits - Повертає кількість збігів для останнього пошуку
  • yaz_itemorder — Підготовка запиту до Z39.50 Item Order з пакетом ILL-Request
  • yaz_present — Готується до пошуку (Z39.50 є)
  • yaz_range — Визначає діапазон записів для вилучення
  • yaz_record — Повертає запис

  • yaz_search — Підготовка пошуку
  • yaz_set_option - Встановлює параметри для з'єднання
  • yaz_sort — Визначає критерій сортування
  • yaz_syntax — Задає перевагу синтаксис для запису
  • yaz_wait — Очікує на виконання запитів Z39.50 серверами
