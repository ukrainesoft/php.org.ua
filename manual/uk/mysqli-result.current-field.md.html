mysqli_result::$current_field

mysqli_field_tell

(PHP 5, PHP 7, PHP 8)

mysqli_result::$current_field -- mysqli_field_tell — Отримує зсув вказівника по відношенню до поточного поля

Опис

Об'єктно-орієнтований стиль

int $mysqli_result->current_field;

Процедурний стиль

mysqli_field_tell(mysqli_result $result): int

Повертає позицію покажчика поля, що використовується під час останнього виклику mysqli_fetch_field(). Це значення може бути використано як аргумент для mysqli_field_seek().

Список параметрів

result Тільки для процедурного стилю: об'єкт mysqli_result, отриманий за допомогою mysqli_query(), mysqli_store_result(), mysqli_use_result() або mysqli_stmt_get_result().

Значення, що повертаються

Повертає поточне усунення вказівника поля.
