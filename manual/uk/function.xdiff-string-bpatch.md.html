#xdiff_string_bpatch

(PECL xdiff >= 1.5.0)

xdiff_string_bpatch — Застосування бінарного патча до рядка

Опис

xdiff_string_bpatch(string $str, string $patch): string

Застосовує до рядка str бінарний патч patch. Ця функція приймає патчі створені як xdiff_string_bdiff(), так і xdiff_string_rabdiff().

Список параметрів

str Оригінальний бінарний рядок.

patch Рядок з бінарним патчем.

Значення, що повертаються

Повертає змінений рядок, або false у разі виникнення помилки.

Дивіться також

  • xdiff_string_bdiff() - Створити бінарний патч для двох рядків
  • xdiff_string_rabdiff() - Порівняти два рядки і створити бінарний патч використовуючи поліномінальний алгоритм Rabin fingerprint
