stripos

(PHP 5, PHP 7, PHP 8)

stripos — Повертає позицію першого входження підрядку без урахування регістру

Опис

stripos(string $haystack, string $needle, int $offset = 0): int|false

Шукає позицію першого входження підрядки needle у рядку haystack.

На відміну від strpos(), ця функція не враховує регістр символів.

Список параметрів

haystack Рядок, в якому проводиться пошук.

needle Зверніть увагу, що needle може містити рядок з одного або більше символів.

До PHP 8.0.0 якщо параметр needle не є рядком, він перетворюється на ціле число і сприймається як код символу. Ця поведінка застаріло з PHP 7.3.0, і покладатися на нього не рекомендується. В Залежно від передбачуваної поведінки, параметр needle має бути або явно приведений до рядка, або має бути виконаний явний виклик chr().

offset Якщо цей параметр вказано, пошук буде розпочато із зазначеної кількості символи з початку рядка. Якщо встановлено негативне значення, відлік позиції початку пошуку буде проведено з кінця рядка.

Значення, що повертаються

Повертає позицію, в якій знаходиться рядок, що шукається початку рядка haystack (незалежно від усунення (offset)). Також Зверніть увагу на те, що позиція рядка відраховується від 0, а не від

Повертає false, якщо шуканий рядок не знайдено.

Увага

Ця функція може повертати як логічне значення false, так і значення не типу boolean, яке наводиться до false. Більше Детальнішу інформацію зверніться до розділу Булев тип. Використовуйте оператор === для перевірки значення, повертається цією функцією.

Список змін

Версія Опис
8.0.0 Передача цілого числа (int) у needle більше не підтримується.
7.3.0 Передача цілого числа (int) у needle оголошена застарілою.
7.1.0 Додано підтримку негативних значень offset.
