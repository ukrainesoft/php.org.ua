# exp

(PHP 4, PHP 5, PHP 7, PHP 8)

exp — Обчислює рівень числа **`e`**

### Опис

**exp**(float `$num`): float

Повертає **`e`** у ступені `num`.

> **Примітка**:
>
> Число **`e`** - основа натурального логарифму. Приблизно одно
>2.718282.

### Список параметрів

`num`
Ступінь для зведення

### Значення, що повертаються

'e' зводиться у ступінь `num`

";echo exp(5.7);?> `
