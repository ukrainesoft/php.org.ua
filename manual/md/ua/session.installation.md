## Установка

Підтримка механізму сесій включена до PHP за замовчуванням. Якщо необхідно
зібрати PHP без такої підтримки, потрібно вказати опцію
**--disable-session** під час конфігурації PHP. Щоб використати
квотування пам'яті, що розділяється (mm) для зберігання даних сесії
необхідно налаштувати PHP з опцією **--with-mm\[u003dDIR\]** .

Версія PHP для Windows має вбудовану підтримку цього модуля. Це
означає, що для використання цих функцій не потрібне завантаження
жодних додаткових модулів.

> **Примітка**:
>
> За замовчуванням усі дані, пов'язані з конкретною сесією, будуть
> зберігатися у файлі директорії, вказаної в опції session.save_path
> файл конфігурації. Файл створюється для кожної сесії (незалежно від
> наявності пов'язаних із нею даних). Це з тим, що сесія відкрита
> (створюється файл), але жодні дані ще записані. Слід звернути
> увага, що така поведінка є побічним ефектом обмежень у
> роботи з файловою системою і цілком можливо, що користувальницький
> обробник сесії (наприклад, той, що використовує базу даних) не
> відстежує сесії, де немає даних.
