Pool::__construct - Створює новий пул воркерів

Опис

public Pool::__construct(int $size, string $class = ?, array $ctor = ?)

Створює новий пул робітників. Пули ліниво створюють свої потоки, що означає, що нові потоки будуть створюватися лише тоді, коли вони необхідні виконання завдань.

Список параметрів

size Максимальна кількість воркерів, яка може створити цей пул

class Клас для нових воркерів. Якщо клас не вказано, то за замовчуванням використовується клас Worker.

ctor Масив аргументів передачі конструкторам нових воркерам.
