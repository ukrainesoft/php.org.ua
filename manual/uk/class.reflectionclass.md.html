Клас ReflectionClass

(PHP 5, PHP 7, PHP 8)

Вступ

Клас ReflectionClass повідомляє інформацію про клас.

Огляд класів

class ReflectionClass implements Reflector {

/* Константи */

const int IS_IMPLICIT_ABSTRACT = 16;

const int IS_FINAL = 64;

Властивості

name Назва класу. Доступно тільки для читання та викидає виняток ReflectionException під час спроби запису.

Зумовлені константи

Модифікатори ReflectionClass

ReflectionClass::IS_IMPLICIT_ABSTRACT Вказує, що клас є абстрактним, тому що він містить абстрактних методів.

ReflectionClass::IS_EXPLICIT_ABSTRACT Вказує, що клас є абстрактним, тому що так вказано при його опис.

ReflectionClass::IS_FINAL Вказує, що клас є остаточним (final)
