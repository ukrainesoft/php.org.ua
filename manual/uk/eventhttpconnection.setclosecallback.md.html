(PECL event >= 1.8.0)

EventHttpConnection::setCloseCallback — Встановлює callback-функцію при закритті з'єднання

Опис

public EventHttpConnection::setCloseCallback( callable $callback , mixed $data = ?): void

Встановлює функцію callback при закритті з'єднання.

Список параметрів

callback Встановлює callback-функцію при закритті з'єднання, що має відповідати прототипу:

callback( EventHttpConnection $conn = null , mixed $arg = null ): void

Значення, що повертаються

Функція не повертає значення після виконання.
