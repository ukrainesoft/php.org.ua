Основна інформація про методи mysqli

MySQL_STMT
ООП-інтерфейс Процедурний інтерфейс Синонім (не використовуйте!) Опис
Властивості
$mysqli_stmt::affected_rows mysqli_stmt_affected_rows() - Повертає число рядків, змінених, віддалених або вставлених під час виконання останньої інструкції
$mysqli_stmt::errno mysqli_stmt_errno() - Повертає код помилки останнього виконаного запиту
$mysqli_stmt::error mysqli_stmt_error() - Повертає рядковий опис помилки останнього запиту
$mysqli_stmt::field_count mysqli_stmt_field_count() - Повертає кількість полів у заданому запиті – не документовано
$mysqli_stmt::insert_id mysqli_stmt_insert_id() - Отримує ID, згенероване останньою операцією INSERT
$mysqli_stmt::num_rows mysqli_stmt_num_rows() - Повертає кількість рядків у результаті виконання запиту
$mysqli_stmt::param_count mysqli_stmt_param_count() Н/Д Повертає кількість параметрів для заданого запиту
$mysqli_stmt::sqlstate mysqli_stmt_sqlstate() - Повертає помилку SQLSTATE на останній виконаний запит
Методи
mysqli_stmt::attr_get() mysqli_stmt_attr_get() - Використовується для отримання поточного атрибуту запиту
mysqli_stmt::attr_set() mysqli_stmt_attr_set() - використовується для модифікації поведінки підготовленого запиту
mysqli_stmt::bind_param() mysqli_stmt_bind_param() Н/Д Передається змінні на підготовлений запит як параметри
mysqli_stmt::bind_result() mysqli_stmt_bind_result() Н/Д Передає змінні до підготовленого запиту для збереження результатів
mysqli_stmt::close() mysqli_stmt_close() - Закриває підготовлений запит
mysqli_stmt::data_seek() mysqli_stmt_data_seek() - Виконує пошук довільного рядка у результатах запиту
mysqli_stmt::execute() mysqli_stmt_execute() mysqli_execute() Виконує підготовлений запит
mysqli_stmt::fetch() mysqli_stmt_fetch() Н/Д Вибір результатів підготовленого запиту у зв'язані змінні
mysqli_stmt::free_result() mysqli_stmt_free_result() -
mysqli_stmt::get_result() mysqli_stmt_get_result() - Отримує результат підготовленого запиту. Доступно лише з модулем mysqlnd.
mysqli_stmt::get_warnings() mysqli_stmt_get_warnings() - Чи не документовано
mysqli_stmt::more_results() mysqli_stmt_more_results() - Перевіряє наявність додаткових результатів запиту у разі множини
mysqli_stmt::next_result() mysqli_stmt_next_result() - Читає наступний результат із множинного запиту
mysqli_stmt::num_rows() mysqli_stmt_num_rows() - Дивіться також $mysqli_stmt::num_rows
mysqli_stmt::prepare() mysqli_stmt_prepare() - Підготовка SQL запиту для запуску
mysqli_stmt::reset() mysqli_stmt_reset() - Скидає підготовлений запит
mysqli_stmt::result_metadata() mysqli_stmt_result_metadata() Н/Д Повертає метадані результатів підготовленого запиту
mysqli_stmt::send_long_data() mysqli_stmt_send_long_data() Н/Д Передача даних блоками
mysqli_stmt::store_result() mysqli_stmt_store_result() - Передає результат із підготовленого запиту

Основна інформація про методи mysqli_stmt

mysqli_result
ООП-інтерфейс Процедурний інтерфейс Синонім (не використовуйте!) Опис
Властивості
$mysqli_result::current_field mysqli_field_tell() - Отримати усунення вказівника на результат для поточного поля
$mysqli_result::field_count mysqli_num_fields() - Отримати кількість полів у результаті
$mysqli_result::lengths mysqli_fetch_lengths() - Повертає довжини стовпців поточного рядка у наборі результатів
$mysqli_result::num_rows mysqli_num_rows() - Отримує число рядків у результаті
Методи
mysqli_result::data_seek() mysqli_data_seek() - Встановлює покажчик результату на довільний рядок у результаті
mysqli_result::fetch_all() mysqli_fetch_all() - Отримує всі результуючі рядки та повертає масив з рядками у вигляді асоціативного масиву та/або нумерованого масиву. Доступно лише з модулем mysqlnd.
mysqli_result::fetch_array() mysqli_fetch_array() - Отримує результуючий рядок у вигляді асоціативного масиву та/або нумерованого масиву
mysqli_result::fetch_assoc() mysqli_fetch_assoc() - Отримує результуючий рядок у вигляді асоціативного масиву
mysqli_result::fetch_column() mysqli_fetch_column() N/A Отримує один стовпець наступного рядка набору результатів
mysqli_result::fetch_field_direct() mysqli_fetch_field_direct() - Отримує метадані одного поля
mysqli_result::fetch_field() mysqli_fetch_field() - Повертає з результату наступне поле
mysqli_result::fetch_fields() mysqli_fetch_fields() - Повертає масив об'єктів із полів результату
mysqli_result::fetch_object() mysqli_fetch_object() - Повертає поточний рядок результату як об'єкт
mysqli_result::fetch_row() mysqli_fetch_row() - Отримує рядок результату як нумерований масив
mysqli_result::field_seek() mysqli_field_seek() - Встановлює усунення вказівника результату на певне поле
mysqli_result::free(), mysqli_result::close, mysqli_result::free_result mysqli_free_result() - Визволяє пам'ять, пов'язану з результатом
