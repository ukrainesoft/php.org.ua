# UI\Area::onKey

(UI 0.9.9)

UI\Area::onKey — Функція зворотного виклику під час натискання

### Опис

protected **UI\Area::onKey**(string `$key`, int `$ext`, int `$flags`)

Виконується при натисканні клавіші

### Список параметрів

`key`
Натиснута клавіша

`ext`
Натиснута розширена кнопка

`flags`
Модифікатори події
