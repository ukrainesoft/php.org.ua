Функції SSH2

Зміст

  • ssh2_auth_agent - Аутентифікація через SSH за допомогою агент ssh
  • ssh2_auth_hostbased_file - Аутентифікація з використанням відкритого ключа хоста
  • ssh2_auth_none — Аутентифікація як "none"
  • ssh2_auth_password Аутентифікація через SSH за допомогою звичайного пароля
  • ssh2_auth_pubkey_file - Аутентифікація з відкритим ключем
  • ssh2_connect — Підключення до SSH-серверу
  • ssh2_disconnect — Закрити з'єднання з віддаленим сервером SSH
  • ssh2_exec — Виконання команди на віддаленому сервері
  • ssh2_fetch_stream — Отримання розширеного потоку даних
  • ssh2_fingerprint — Отримання відбиток віддаленого сервера
  • ssh2_forward_accept - Приймає з'єднання, створене слухачем
  • ssh2_forward_listen - Зв'язує порт на віддаленому сервері та прослуховує з'єднання
  • ssh2_methods_negotiated - Повертає список узгоджених методів
  • ssh2_poll - Опитує канали/слухачів/потоки щодо подій
  • ssh2_publickey_add — Додає авторизований відкритий ключ
  • ssh2_publickey_init - Ініціалізує підсистему відкритого ключа
  • ssh2_publickey_list - Список вже авторизованих відкритих ключів
  • ssh2_publickey_remove - Видаляє авторизований відкритий ключ
  • ssh2_scp_recv — Запит файлу через SCP
  • ssh2_scp_send — Надсилання файлу через SCP
  • ssh2_send_eof - Відправляє EOF в потік
  • ssh2_sftp_chmod — Зміна прав доступу
  • ssh2_sftp_lstat — Інформація про символічне посилання
  • ssh2_sftp_mkdir — Створити директорію
  • ssh2_sftp_readlink - Повертає об'єкт за символічним посиланням
  • ssh2_sftp_realpath — Визначає повний шлях по даному рядку зі шляхом
  • ssh2_sftp_rename — Перейменовує файл
  • ssh2_sftp_rmdir — Видаляє директорію
  • ssh2_sftp_stat — Інформація про файл
  • ssh2_sftp_symlink — Створити символічне посилання
  • ssh2_sftp_unlink — Видалити файл на сервері
  • ssh2_sftp - Ініціалізувати підсистему SFTP
  • ssh2_shell — запитує інтерактивний термінал
  • ssh2_tunnel — Відкрити тунель через віддалений сервер
