ReflectionFunction::__construct

(PHP 5, PHP 7, PHP 8)

ReflectionFunction::__construct - Конструктор класу ReflectionFunction

Опис

public ReflectionFunction::__construct(Closure|string $function)

Створює об'єкт класу ReflectionFunction.

Список параметрів

function Ім'я функції відображення або замикання.

Помилки

Об'єкт класу ReflectionException, якщо аргумент function не містить допустимої функції.
