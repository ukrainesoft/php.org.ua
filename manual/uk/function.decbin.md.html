decbin

(PHP 4, PHP 5, PHP 7, PHP 8)

decbin — Переводить число з десяткової системи числення до двійкової

Опис

decbin(int $num): string

Повертає рядок, що містить двійкове подання зазначеного аргументу num.

Дивіться також

  • bindec() - Двійковий до десяткового
  • decoct() - Перекладає число з десяткового системи числення у вісімкову
  • dechex() - Перекладає число з десяткового системи числення в шістнадцяткову
  • base_convert() - Перетворення числа між довільними системами числення
  • printf() - Виводить відформатований рядок, використовуючи формати '%b', '%032b' або '%064b'
  • sprintf(), використовуючи формати %b, %032b або %064b
