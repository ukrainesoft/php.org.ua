Клас PDO

(PHP 5 = 5.1.0, PHP 7, PHP 8, PECL pdo = 0.1.0)

Вступ

З'єднання між PHP і сервером бази даних.

Огляд класів

class PDO {

/* Методи */

public __construct( string $dsn, ?string $username = null, ?string $password = null, ?array $options = null )

public beginTransaction(): bool

public commit(): bool

public errorCode(): ?string

public errorInfo(): array

public exec(string $statement): int|false

public getAttribute(int $attribute): mixed

public static getAvailableDrivers(): array

public inTransaction(): bool

public lastInsertId(?string $name = null): string|false

public prepare(string $query, array $options = []): PDOStatement|false

public query(string $query, ?int $fetchMode = null): PDOStatement|false

public query(string $query, ?int $fetchMode = PDO::FETCH_COLUMN, int $colno): PDOStatement|false

public query( string $query, ?int $fetchMode = PDO::FETCH_CLASS, string $classname, array $constructorArgs ): PDOStatement|false

public query(string $query, ?int $fetchMode = PDO::FETCH_INTO, object $object): PDOStatement|false

public quote(string $string, int $type = PDO::PARAM_STR): string|false

public rollBack(): bool

public setAttribute(int $attribute, mixed $value): bool

}

Зміст

  • PDO::beginTransaction - Ініціалізація транзакції
  • PDO::commit - Фіксує транзакцію
  • PDO::__construct - Створює екземпляр PDO, що надає з'єднання з базою даних
  • PDO::errorCode — Повертає код SQLSTATE результату останньої операції з базою даних
  • PDO::errorInfo — Отримує розширену інформацію про помилку, що сталася під час останнього звернення до базі даних
  • PDO::exec — Виконує SQL-запит та повертає кількість порушених рядків
  • PDO::getAttribute — Отримати атрибут з'єднання з базою даних
  • PDO::getAvailableDrivers - Повертає масив доступних драйверів PDO
  • PDO::inTransaction — Перевіряє, чи розпочато транзакція
  • PDO::lastInsertId - Повертає ID останньої вставленого рядка або значення послідовності
  • PDO::prepare — Підготовка запиту до виконання та повертає пов'язаний з цим запитом об'єкт
  • PDO::query — Підготовка та виконання виразу SQL без наповнювачів
  • PDO::quote — Укладає рядок у лапки для використання у запиті
  • PDO::rollBack - Відкат транзакції
  • PDO::setAttribute — Встановлення атрибута
