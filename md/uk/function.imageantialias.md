#imageantialias

(PHP 4 \>u003d 4.3.2, PHP 5, PHP 7, PHP 8)

imageantialias — Чи потрібно застосовувати функції згладжування чи ні

### Опис

**imageantialias**([GdImage](class.gdimage.md) `$image`, bool
`$enable`): bool

Задіює механізми згладжування при малюванні ліній та замкнутих
багатокутників. Не підтримує компоненти альфа. Використовує операції
прямого змішування кольорів. Працює тільки з truecolor-зображеннями.

Стилі та товщина не підтримуються.

Використання згладжених примітивів разом із прозорим тлом може
дати несподіваний результат. Метод змішування використовує колір тла
нарівні з будь-яким іншим кольором. Відсутність підтримки альфа компонента не
дозволяє застосовувати заснований у ньому метод згладжування.

### Список параметрів

`image`
Об'єкт [GdImage](class.gdimage.md), який повертається однією з функцій
створення зображень, наприклад, такий як
[imagecreatetruecolor()](function.imagecreatetruecolor.md).

`enable`
Увімкнути згладжування чи ні.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у
у разі виникнення помилки.
