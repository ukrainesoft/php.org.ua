SimpleXMLElement::asXML

(PHP 5, PHP 7, PHP 8)

SimpleXMLElement::asXML — Повертає сформований XML-документ у вигляді рядки на основі елемента SimpleXML

Опис

public SimpleXMLElement::asXML(?string $filename = null): string|bool

Метод asXML задає формат даних батьківських об'єктів у версії XML 1.0.

Список параметрів

filename Якщо вказано значення у вигляді рядка (string), функція запише дані у файл, а чи не поверне їх.

Значення, що повертаються

Якщо filename не вказано, то функція поверне рядок (string) у разі успішного виконання та false у разі виникнення помилки. Якщо параметр вказано, то функція поверне true, якщо файл буде успішно записаний і false інакше.

Список змін

Версія Опис
8.0.0 filename тепер припускає значення null.
