ReflectionParameter::__construct

(PHP 5, PHP 7, PHP 8)

ReflectionParameter::__construct — Конструктор

Опис

public ReflectionParameter::__construct(string|array|object $function, int|string $param)

Створює екземпляр ReflectionParameter.

Список параметрів

function Функція, яку потрібно відобразити.

param Або ціле число (int), що вказує позицію параметра (починаючи з нуля), або ім'я параметра як рядка (string).
