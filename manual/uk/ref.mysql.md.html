Функції СУБД MySQL

Примітки

Примітка:

Більшість функцій MySQL приймають link_identifier як останній, Опціональний параметр. Якщо він не вказаний, то використовується останнє Відкрите з'єднання. Якщо з'єднань немає, то модуль намагається відкрити з'єднання використовуючи параметри, вказані в php.ini. В разі Невдачі, функції повертають false. false.

Зміст

  • mysql_affected_rows - Повертає число порушених минулою операцією рядів
  • mysql_client_encoding - Повертає кодування з'єднання
  • mysql_close — Закриває з'єднання з сервером MySQL
  • mysql_connect — Відкриває з'єднання з сервером MySQL
  • mysql_create_db - Створює базу даних MySQL
  • mysql_data_seek — Переміщує внутрішній покажчик в результаті запиту
  • mysql_db_name — Повертає назву бази даних із виклику до mysql_list_dbs
  • mysql_db_query — Перемикається на вказану базу даних та надсилає запит
  • mysql_drop_db — Знищує базу даних MySQL
  • mysql_errno — Повертає чисельний код помилки виконання останньої операції з MySQL
  • mysql_error — Повертає текст помилки останньої операції з MySQL
  • mysql_escape_string - Екранує рядок для використання у mysql_query
  • mysql_fetch_array - Обробляє ряд результату запиту, повертаючи асоціативний масив, чисельний масив або обидва
  • mysql_fetch_assoc - Повертає ряд результату запиту як асоціативний масив
  • mysql_fetch_field - Повертає інформацію про колонку з результату запиту як об'єкта
  • mysql_fetch_lengths - Повертає довжину кожного поля в результаті
  • mysql_fetch_object - Обробляє ряд результату запиту та повертає об'єкт
  • mysql_fetch_row - Обробляє ряд результату запиту та повертає масив з числовими індексами
  • mysql_field_flags - Повертає прапори, пов'язані із зазначеним полем результату запиту
  • mysql_field_len — Повертає довжину вказаного поля
  • mysql_field_name — Повертає назва вказаної колонки результату запиту
  • mysql_field_seek - Встановлює внутрішній покажчик результату на передане усунення поля
  • mysql_field_table - Повертає назва таблиці, якій належить зазначене поле
  • mysql_field_type — Повертає тип вказаного поля з результату запиту
  • mysql_free_result — Звільняє пам'ять від результату запиту
  • mysql_get_client_info - Повертає дані про MySQL-клієнт
  • mysql_get_host_info - Повертає інформацію про з'єднання з MySQL
  • mysql_get_proto_info - Повертає інформацію про протокол MySQL
  • mysql_get_server_info - Повертає інформацію про сервер MySQL
  • mysql_info — Повертає інформацію про останньому запиті
  • mysql_insert_id - Повертає ідентифікатор, згенерований при останньому INSERT-запиті
  • mysql_list_dbs — Повертає список баз даних, доступних на сервері
  • mysql_list_fields - Повертає список колонок таблиці
  • mysql_list_processes - Повертає список процесів MySQL
  • mysql_list_tables - Повертає список таблиць бази даних MySQL
  • mysql_num_fields - Повертає кількість полів результату запиту
  • mysql_num_rows - Повертає кількість рядів результату запиту
  • mysql_pconnect — Встановлює постійне з'єднання з сервером MySQL
  • mysql_ping — Перевіряє з'єднання з сервером і переєднується за необхідності
  • mysql_query — Надсилає запит MySQL
  • mysql_real_escape_string - Екранує спеціальні символи в рядках для використання в виразах SQL
  • mysql_result — Повертає дані результату запиту
  • mysql_select_db - Вибирає базу даних MySQL
  • mysql_set_charset - Встановлює кодування клієнта
  • mysql_stat — Повертає поточний статус сервера
  • mysql_tablename — Повертає ім'я таблиці, що містить вказане поле
  • mysql_thread_id - Повертає ідентифікатор потоку
  • mysql_unbuffered_query - Надсилає запит MySQL без авто-обробки результату та його буферизації
