DateTimeZone::listAbbreviations

#timezone_abbreviations_list

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

DateTimeZone::listAbbreviations -- timezone_abbreviations_list Повертає асоціативний масив, який містить прапор переходу на літнє час, зсув та ім'я часового поясу

Опис

Об'єктно-орієнтований стиль

public static DateTimeZone::listAbbreviations(): array

Процедурний стиль

timezone_abbreviations_list(): array

Список абревіатур, що повертається, включає всі історичні випадки використання абревіатур, що може призвести до правильних, але заплутаним записам. Існують також протиріччя, оскільки "pst" використовується як у США, так і на Філіппінах.

Тому список, який повертає цю функцію, не підходить для створення масиву з опціями для надання вибору часового поясу користувачам.

Примітка:

Дані для функції попередньо компілюються з міркувань продуктивності та не оновлюються при використанні більш нової версії » timezonedb.

Список параметрів

Ця функція не має параметрів.

Значення, що повертаються

Повертає масив скорочень часових поясів.

Приклади

Приклад #1 Приклад використання timezone_abbreviations_list()

<?php$timezone_abbreviations = DateTimeZone::listAbbreviations();print_r($timezone_abbreviations["acst"]);?>

Результатом виконання цього прикладу буде щось подібне:
