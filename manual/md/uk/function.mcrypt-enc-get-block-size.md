# mcrypt_enc_get_block_size

(PHP 4 \>u003d 4.0.2, PHP 5, PHP 7 \< 7.2.0, PECL mcrypt \>u003d 1.0.0)

mcrypt_enc_get_block_size — Повертає розмір блоку алгоритму

**Увага**

Ця функція оголошена *УСТАРНІЙ*, починаючи з PHP 7.1.0 і була *Видалена*
у версії PHP 7.2.0. Використовувати цю функцію не рекомендується.

### Опис

**mcrypt_enc_get_block_size**(resource `$td`): int

Повертає розмір блоку алгоритму.

### Список параметрів

`td`
Дескриптор шифрування.

### Значення, що повертаються

Повертає розмір блоку алгоритму у байтах.
