#cubrid_get_server_info

(PECL CUBRID = 8.3.0)

cubrid_get_server_info — Повертає версію сервера CUBRID

Опис

cubrid_get_server_info(resource $conn_identifier): string

Функція повертає рядок, який представляє версію сервера CUBRID.

Список параметрів

conn_identifier З'єднання CUBRID.

Значення, що повертаються

Рядок, що представляє версію сервера CUBRID у разі успішного виконання, або false у разі виникнення помилки.
